Endpoints

All endpoints are static JSON files, auto-generated by GitHub Actions on every commit. Fetch via HTTP GET.

📊 Countries v1.2

GET https://kfat77.github.io/digital-nomad-cn/api/countries.json

Full structured data for 60 countries, including cost of living, safety, internet speed, visa policies, and deep-dive guide fields (25+ fields).

Core fields: id, name, name_en, region, digital_nomad_score, cost_of_living_index, safety_index, internet_speed_avg_mbps, china_passport_visa_status, has_digital_nomad_visa, climate_zone, timezone, dn_community_size, median_stay_months

Deep-dive fields (v1.2): healthcare_quality, english_proficiency, monthly_rent_usd_low/high, food_cost_usd_daily, transport_cost_usd_monthly, coworking_spaces_count, local_tax_rate, pros, cons, ideal_for, best_neighborhoods

🏙️ Cities

GET https://kfat77.github.io/digital-nomad-cn/api/cities.json

Full data for 122 digital nomad cities, including coworking, climate, English level, community size, remote work friendly score.

Key fields: id, name, name_en, country_id, country_name, digital_nomad_score, cost_of_living_index, rent_index, safety_index, internet_speed_avg_mbps, coworking_spaces, coworking_score, climate, avg_temp_c, rainy_season, english_level, chinese_community_size, neighborhoods, food_scene, nightlife, transportation, walkability, bike_friendly, description, remote_work_friendly_score, monthly_cost_usd_low, monthly_cost_usd_high

Code Examples

No API key required. Just GET. Examples below show how to call the data API.

// Get all countries and filter high-score destinations fetch('https://kfat77.github.io/digital-nomad-cn/api/countries.json') .then(r => r.json()) .then(data => { // Filter: score >= 80 and cost <= 60 const topCountries = data.data .filter(c => c.digital_nomad_score >= 80 && c.cost_of_living_index <= 60) .sort((a, b) => b.digital_nomad_score - a.digital_nomad_score); console.log(topCountries.map(c => `${c.name_en}: ${c.digital_nomad_score}`)); });
// Get stats summary fetch('https://kfat77.github.io/digital-nomad-cn/api/stats.json') .then(r => r.json()) .then(stats => { console.log(`Avg nomad score: ${stats.countries.avg_digital_nomad_score}`); console.log(`Visa-free count: ${stats.countries.by_visa_status.visa_free || 0}`); });
# Get country data and filter (Python example) import requests url = "https://kfat77.github.io/digital-nomad-cn/api/countries.json" resp = requests.get(url) data = resp.json() # Filter score >= 80 top = [c for c in data["data"] if c["digital_nomad_score"] >= 80] for c in sorted(top, key=lambda x: x["digital_nomad_score"], reverse=True): print(f"{c['name_en']}: {c['digital_nomad_score']} | Cost: {c['cost_of_living_index']}")
# Get countries curl -s "https://kfat77.github.io/digital-nomad-cn/api/countries.json" | python -m json.tool # Get cities curl -s "https://kfat77.github.io/digital-nomad-cn/api/cities.json" | python -m json.tool # Get stats curl -s "https://kfat77.github.io/digital-nomad-cn/api/stats.json" | python -m json.tool
