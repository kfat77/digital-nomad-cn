Milestone 1: Data Infrastructure v1.0

  • 21 种Digital NomadVisas信息

已完成 2025 Q2

Milestone 2: SEO 基础设施

  • robots.txt + sitemap.xml（554 URL）

已完成 2025 Q2

Milestone 3: 结构化数据 API v1.3.0

  • api/manifest.json — AI Agent 发现端点
  • api/schema.json — Schema Registry

已完成 2025 Q3

Milestone 4: 国际化 i18n 基础

已完成 2025 Q3

Milestone 5: Core Web Vitals 性能优化

  • requestIdleCallback 延迟非关键任务
