## 核心设计原则

```
┌─────────────────────────────────────────────────────────┐
│  1. Data First                                          │
│     数据是核心资产，网站只是消费者之一                    │
│                                                         │
│  2. API First                                           │
│     所有数据通过标准化API暴露，支持多种协议               │
│                                                         │
│  3. AI First                                            │
│     设计之初就考虑AI Agent调用，MCP Server是核心能力      │
│                                                         │
│  4. Open Source First                                   │
│     所有Schema和数据开放，社区共同维护                    │
│                                                         │
│  5. Developer First                                     │
│     SDK、CLI、文档、示例，全方位降低开发者门槛            │
│                                                         │
│  6. Single Source of Truth                              │
│     schemas/ 和 datasets/ 是唯一的真相源                  │
│     所有消费者从这里获取数据                             │
│                                                         │
│  7. Schema-Driven                                       │
│     Schema定义一次，自动生成TypeScript/Python/GraphQL     │
│                                                         │
│  8. Edge-First Deployment                               │
│     API部署在全球边缘节点，毫秒级响应                    │
└─────────────────────────────────────────────────────────┘
```
