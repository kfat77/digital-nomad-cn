## API调用链

```
┌─────────────────────────────────────────────────────────────────────────┐
│                         API Call Chain                                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  用户请求                                                                │
│     │                                                                    │
│     ▼                                                                    │
│  ┌─────────────┐                                                        │
│  │ API Gateway │  ← Cloudflare Workers (全球边缘)                        │
│  │             │     1. 认证                                              │
│  │             │     2. 限流                                              │
│  │             │     3. 缓存检查                                          │
│  └──────┬──────┘                                                        │
│         │                                                                │
│         ▼                                                                │
│  ┌─────────────┐                                                        │
│  │   Router    │  ← 根据路径分发                                         │
│  │             │     /v1/countries → CountryHandler                      │
│  │             │     /graphql → GraphQLResolver                          │
│  │             │     /mcp → MCPServer                                    │
│  └──────┬──────┘                                                        │
│         │                                                                │
│         ▼                                                                │
│  ┌─────────────┐                                                        │
│  │ Data Loader │  ← 从datasets/加载数据                                  │
│  │             │     1. 检查KV缓存                                        │
│  │             │     2. 未命中则读取JSON                                  │
│  │             │     3. 写入缓存                                          │
│  └──────┬──────┘                                                        │
│         │                                                                │
│         ▼                                                                │
│  ┌─────────────┐                                                        │
│  │  Response   │  ← 格式化响应                                           │
│  │  Formatter  │     1. 按请求语言过滤                                    │
│  │             │     2. 按权限过滤                                        │
│  │             │     3. 添加metadata                                      │
│  └──────┬──────┘                                                        │
│         │                                                                │
│         ▼                                                                │
│  用户收到响应                                                            │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
```

## AI Agent交互

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        AI Agent Interaction                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  User: "我想去东南亚远程工作3个月，预算1000美元/月，推荐几个地方"          │
│                                                                          │
│     │                                                                    │
│     ▼                                                                    │
│  ┌─────────────────────────────────────────────────────────────────┐    │
│  │                     Claude (with MCP)                            │    │
│  │                                                                  │    │
│  │  Step 1: 解析意图                                                │    │
│  │    → 意图: recommend_destinations                               │    │
│  │    → 约束: {region: "southeast-asia", budget: 1000, duration: 3} │    │
│  │                                                                  │    │
│  │  Step 2: 调用工具                                                │    │
│  │    → nomad_search_countries(                                    │    │
│  │        region: "southeast-asia",                                 │    │
│  │        tags: ["digital-nomad-friendly"],                         │    │
│  │        maxCost: 1000                                             │    │
│  │      )                                                           │    │
│  │                                                                  │    │
│  │  Step 3: 获取详细数据                                            │    │
│  │    → nomad_get_city("chiang-mai")                               │    │
│  │    → nomad_get_city("da-nang")                                  │    │
│  │    → nomad_get_city("penang")                                   │    │
│  │                                                                  │    │
│  │  Step 4: 对比分析                                                │    │
│  │    → nomad_compare_costs(["chiang-mai", "da-nang", "penang"])   │    │
│  │                                                                  │    │
│  │  Step 5: 生成回答                                                │    │
│  │    → 结构化推荐列表                                              │    │
│  │    → 附带数据来源                                                │    │
│  │                                                                  │    │
│  └─────────────────────────────────────────────────────────────────┘    │
│                                                                          │
│     │                                                                    │
│     ▼                                                                    │
│  Response:                                                               │
│  "基于Digital Nomad CN的数据，我为您推荐以下3个目的地："                  │
│  "1. 泰国清迈 - 月生活成本$800-1000，有大量数字游民社区..."               │
│  "2. 越南岘港 - 月生活成本$700-900，海滩城市..."                         │
│  "3. 马来西亚槟城 - 月生活成本$900-1100，中文友好..."                    │
│  "数据来源: https://api.digital-nomad.cn/v1/..."                        │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
```
