显微镜成像并结合图像重建获得大分子复合物近原子分辨率结构
的技术。

蛋白质折叠识别 
蛋白质折叠识别方法又称为穿线法(Threading)，意为把目标序列穿过模板的
三维结构以判断序列与结构的适应度。该种方法的理论支撑是在蛋白质序列
不相似的情况下，蛋白质的结构仍然可能相似，即蛋白质存在弱同源性。 
应用场景：远缘同源蛋白或具有保守折叠但序列差异大的情况。

远源与近同源蛋白 
• 一般序列相似的蛋白，结构也相似，若其具有相同的起源，称为近同源
蛋白。 
 
• 同源蛋白在数亿年的进化过程中，为维持功能的稳定，结构保持相对稳
定；但是其序列会持续发生突变。存在序列非常不相似，但是结构域功
能相似的同源蛋白，称为远源蛋白。 
 
• 肌红蛋白(Myoglobin)和血红蛋白(Hemoglobin)可能是最早发现的远源蛋
白。

从头计算(ab initio folding) 
• 源于Anfinsen的“最低自由能构型假说”。 
 
• 是以物理为基础来研究蛋白质的折叠方法。 
• 在算法中除了依靠多序列比对信息(进化信息)、以及结构片段信息，更多地
依靠接触(contact)信息以及疏水等信息。 
 
 
• 两个关键： 
• 设计适当的能量函数 
• 构建搜索算法(例如模拟退火算法)找到相应的最低自由能对应的构象。 

自由建模(从头预测) 
• 从生物物理化学的角度来讲，蛋白质在自然状态下对应的是自由能最低的状态。 
• 从分子动力学和分子立场的角度来设计合适的能量函数。 
• 类似于迷雾山地下山(“盲人下山”)问题，寻找能量较低的三维结构构象。 

能量最小化方法预测蛋白质三级结构 
原理：天然稳定构像对应自由能最低的构像 
困难：折叠态与非折叠态能量差很小，要求特别准确的能量项，局部能

AlphaFold算法 
• 蛋白质序列 → 多序列比对(MSA) → 特征提取 → Evoformer模块 → 结构模
块(Structure Module) → 3D原子坐标 → 迭代精炼(Iteration) 
• Evoformer模块：同时处理序列进化和空间结构信息、学习空间约束生成注
意力图。Evoformer的结果残基对表示pair_rep包含了丰富的空间信息：(a) 1. 
预测距离分布；(b)预测取向分布(单位向量间的角度)；(c)预测二面角分布(φ, 
ψ, ω) 
• 结构模块：预测每个残基的刚体变换。所有同类型氨基酸的局部几何是固定
的，只有刚体变换不同。结构模块直接输出每个残基的旋转。 
• AlphaFold通过多次迭代逐步改进预测。多轮改进，每轮利用前一轮的信息，
类似能量最小化，但是用神经网络计算。 
• AlphaFold在神经网络中通过自注意力机制自动学习如何将四元数映射为合
理的旋转，这种隐式学习比显式约束更灵活。旋转由数据驱动的统计规律决
定。AlphaFold学习的是“哪种旋转在天然蛋白质中更可能出现”,使用四元
数表示蛋白质主链每个残基的朝向，四元数本身也是被预测的参数。 
 
 
 
 
 
AlphaFold算法的刚体变换 
• 每个残基的刚体变换有6个自由度，虽然旋转有3个自由度，但用四元数表示
需要4个数字(有约束：q_w²+q_x²+q_y²+q_z²=1)。 
• 物理意义限制：蛋白质中只有某些变换是合理的 
• 刚体变换预测器 → 四元数+平移 
• 假设预测第50号残基亮氨酸Leu的变换：步骤1：提取特征 残基50的特征 = 
[0.3, -0.8, 0.5, ...] # 256维向量 含义： - 0.3: 这个位置倾向于在蛋白质内部 - -
0.8: 可能参与疏水相互作用 - 0.5: 可能形成β折叠

AlphaFold刚体变换预测的核心思想 
 • 固定局部，变化全局：每个氨基酸的内部几何固定，只预测如何放置 
• 6自由度预测：3个平移 + 3个旋转(用四元数表示) 
• 迭代精炼：从粗到细，多轮改进 
• 物理约束：通过IPA和损失函数保证合理性 
• 数据驱动：从已知结构中学习变换模式 
• 这种方法的优势在于： 
• 保证了物理合理性(刚体不变形)、减少了搜索空间(从无限可能到6个自由度) 
• 适合神经网络预测(连续、可微) 
• 生成的结构自动满足基本几何约束 
• 这就是为什么AlphaFold能够快速、准确地"直接变出"蛋白质3D结构——实
际上是在预测每个标准氨基酸模块的摆放方式，而不是从头创造每个原子位
置。 
 
晶格模型与AlphaFold模型比较 
• AlphaFold代表了蛋白质结构预测的根本性范式转变： 
• 从搜索到计算：不再搜索构象空间，而是计算最可能结构 
• 从能量最小化到模式识别：用数据模式替代物理能量函数 
• 从专家知识到数据驱动：让网络自己学习约束，而非手工设计 
• 从局部优化到全局优化：端到端训练优化整个流程 
• 这种转变的成功证明了深度学习可以从数据中直接学习复杂的物理和生物规
律，开启了计算结构生物学的新时代。然而，晶格模型提供的物理洞察和可
解释性仍然是重要的补充，两者在未来很可能继续共存和融合。

AlphaFold核心算法 
• 结构模块通常采用循环神经网络风格的更新机制，多次迭代优化CA骨架构
象。每次迭代都会重新计算当前构象下的特征反馈给Evoformer-like子网络，
形成闭环反馈。 
• 将原始PDB坐标转化为固定尺寸的3D张量，对蛋白质结构进行网格化表示。 
• AlphaFold损失函数主要包括：(a)坐标回归损失 ：L1/L2 loss on Cα positions 
compared to ground truth (b)角度损失 ：惩罚主链二面角(φ, ψ)偏离
Ramachandran图允许区域。(c)键长与键角损失 ：约束Cα–C、C–N等共价键
长度接近理想值。(d)碰撞惩罚项 ：避免原子间距离过近。 
损失项 权重 物理意义 
坐标回归损失  1.0 几何准确性 
角度损失  0.5 二级结构合理性 
键长与键角损失  0.3 化学合理性 
碰撞惩罚项  0.2 空间冲突避免

粗粒化模型(C主链)的具体计算步骤 
(1) 起始构象ri，构象发生微小的随机变化(2~9aa)产生新构象ri＋1，相应能量也
发生变化，能量变化为E=Enew-Eold； 
(2) 如果E不大于零 时接受构象变化，新构象成为下一步的初始构象； 
(3)当E大于零 时，则在(0，1)间产生一随机数R，将其与 P(E)=exp(- E/kT)
相比较；若R< P(E)则接受，新构象成为下一步的初始构象。否则拒绝变化，
旧的构象仍然是下一次随机变化的起始点； 
优点：取样构型恰当，对低能量构象取样的几率大 
难点：构造能量函数以及模型随机波动的代码量较大 
缺点：计算量较大，一般一个蛋白需要几十分钟以上时间才能进行较好的模
拟。

侧链预测软件SCWRL 
•  SCWRL 是“Side Chain With Rotamer Library”的缩写，直译就是“使用旋转
异构体库的侧链”。 
• 输入：蛋白质主链的原子坐标文件(只包含N, Cα, C, O等主链原子)。输出：
一个完整的PDB文件，包含所有重建的侧链原子坐标。通常还会提供每个残
基所选择的旋转异构体信息和能量评估。 
• 原理：侧链中的单键可以旋转，但其构象并非完全随机。由于空间位阻和能
量偏好，侧链倾向于停留在少数几个能量较低的、稳定的角度组合上，这些
离散的构象称为 “旋转异构体”。 
• Rotamer 库从大量已知的高分辨率X射线晶体结构中统计得出。例如，一个
亮氨酸残基，其 χ₁ 和 χ₂ 二面角最常出现的组合可能只有3-5种。SCWRL 使
用一个包含所有20种氨基酸常见旋转异构体的离散化构象库。 
• 侧链组装的核心挑战在于 “侧链间的空间冲突”。一个残基最优的旋转异
构体选择，可能被相邻残基的侧链“撞开”。因此，这不是20个独立的优化
问题，而是一个全局优化问题。

AlphaFold2的架构主要由两个核心模块构成，它们内部都使用了注意力机制。 
模块一：Evoformer —— 进化信息的精炼厂 
Evoformer模块的工作是解读输入的两种关键信息，并让它们相互增强。 
 
输入信息： 
 
MSA (Multiple Sequence Alignment，多序列比对) 表示：一个矩阵，记录了目标蛋白及其大量同源序列的演化过程。思想
是：在演化中，如果一个位置变了，与之在3D结构中接触的位置也可能发生补偿性突变，MSA就捕捉了这种“共演化”
信号。 
 
对偶表示 (Pair Representation)：一个矩阵，记录了每对氨基酸之间的潜在关系，如距离、化学键等。 
 
核心注意力机制： 
 
MSA行/列注意力：为了处理MSA信息，Evoformer设计了巧妙的注意力变体。它对MSA矩阵的行(不同序列)和列(同一位
点在不同序列中的演化模式)分别应用注意力，以提取演化约束信息。 
 
三角注意力 (Triangular Attention)：这是AlphaFold的关键创新。它的作用是强制模型满足几何约束。比如，三点间距离必
须满足“两边之和大于第三边”的三角不等式。三角注意力通过对(i, k)和(k, j)对的关系进行自注意力，来更新(i, j)对的
表示，从而确保最终预测的3D结构是几何自洽的。 
 
通过这几十层(48层)的复杂更新，MSA中的共演化信息被提炼出来，用于丰富对偶表示，最终生成富含几何约束的配对
信息。

模块二：结构模块 (Structure Module) —— 3D结构的构建师 
当Evoformer完成信息处理后，结构模块接手，负责在3D空间中“搭建”蛋白质。 
 
残基气体表示：它把每个氨基酸残基看作一个能在空间中独立平移和旋转的刚体（Residue Gas），初始状态就像一堆漂

不变点注意力 (Invariant Point Attention, IPA)：这是结构模块的核心。IPA在Evoformer生成的对偶表示指导下，计算每个
“残基刚体”之间的空间注意力。它的一个关键特性是不变性 (Invariant)，即对蛋白质整体的平移或旋转保持不变，确保
模型专注于学习其内在的相对结构。 
 
通过IPA的迭代更新，这些漂浮的残基逐步找到各自正确的空间位置，最终组装成一个完整的、高精度的3D结构。

迭代回收 (Recycling)：模型并非一次到位。它会将上一轮预测出的结构信息重新作为输入，送回网络进行多轮（通常3轮）

# 简化的Evoformer迭代 
class EvoformerIteration: 
    def __call__(self, msa, pair): 
        # 1. MSA行注意力 - 每条序列内寻找关联残基 
        msa = row_attention_with_pair_bias(msa, pair) 
         
        # 2. MSA列注意力 - 不同序列间同一位置的共变信号 
        msa = column_attention(msa) 
         
        # 3. 外积取平均 - 从MSA提取pair关系 
        pair += outer_product_mean(msa) 
         
        # 4. 三角更新 - 强制满足几何约束 
        pair = triangle_multiplication_outgoing(pair) 
        pair = triangle_multiplication_incoming(pair) 
         
        # 5. 三角注意力 - 传播pair信息 
        pair = triangle_attention_starting_node(pair) 
        pair = triangle_attention_ending_node(pair) 
         
        # 6. MSA过渡层 
        msa = transition(msa) 
         
        return msa, pair

def triangle_multiplication_outgoing(pair): 
    """强制满足几何约束：d_ij + d_jk >= d_ik""" 
    # 沿着路径j->k对信息进行聚合 
    # 这是AlphaFold能够产生物理合理结构的关键机制之一 
    left = layer_norm(pair) 
    right = layer_norm(pair) 
     
    # 外积并沿残差连接 
    gating = sigmoid(linear(gate)) 
    update = einsum('ijc, jkc -> ikc', left, right) 
    update = layer_norm(update) 
    update = gating * update 
     
    return pair + linear(update)

# 简化的Structure Module 
class StructureModule: 
    def __init__(self, num_iterations=8): 
        self.num_iterations = num_iterations 
     
    def __call__(self, single_repr, pair_repr): 
        # 1. 初始化所有残基为刚体(平移+旋转) 
        rigids = Rigid.from_tensor(initial_coords) 
         
        for i in range(self.num_iterations): 
            # 2. IPA - 不变点注意力 
            attention = invariant_point_attention(single_repr, pair_repr, rigids) 
             
            # 3. 更新刚体变换 
            update = self.rigid_update(attention) 
            rigids = rigids.compose(update) 
             
            # 4. 更新单表征 
            single_repr = transition(attention) 
         
        # 5. 计算侧链扭转角 
        sidechain_torsions = self.sidechain_predict(single_repr) 
         
        return {'positions': rigids, 'sidechains': sidechain_torsions}

侧链预测的二面角方法 
AlphaFold不直接预测每个原子的坐标，而是先预测侧链的"
扭转角" (Dihedral Angles)。一个氨基酸通常有χ1、χ2、χ3、

几何约束，计算出每个原子的最终位置。这种方法比直接回
归原子坐标更稳定、更符合化学规则。

晶格模型是蛋白质结构预测中的一种简化表示方法，它将蛋白质

简单的结构和能量准则，就能从海量构象中筛选出类天然结构，
而无需预先知道任何天然结构的细节。

全链构象 = 向量序列(r1,r2,…,rn), r1 ≠ r2
