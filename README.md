## 🌍 项目介绍

Digital Nomad CN 致力于构建一个开放的中文 Global Mobility 数据平台。

通过结构化数据、自动验证和静态网站，为用户提供全球流动信息。

帮助中文用户探索：

- 💳 海外银行卡与支付工具
- 📱 全球通信与 eSIM
- 🌍 签证与长期居留
- 🏙️ 数字游民城市指南
- 📊 Global Mobility 数据

```text

│

```

## 🚀 本地运行

启动网站：

```powershell
python -m http.server 4173 --directory docs
```

打开：

```
http://localhost:4173
```
