## 📖 项目介绍

Digital Nomad CN 是一个面向中国用户的数字游民信息整理网站。

目标是帮助用户更方便地了解海外生活相关信息，包括：

- 💳 海外银行卡与支付工具
- 📱 海外电话卡与 eSIM
- 📈 海外金融工具与证券平台信息
- 🌍 出境生活资料整理


---

## ✨ 内容模块


### 💳 海外银行卡

整理海外银行卡相关信息：

- 开户条件
- 使用场景
- 注意事项


### 📱 全球通信

整理：

- 海外电话卡
- eSIM 服务
- 网络连接方案


### 📈 海外金融工具

整理：

- 海外金融平台信息
- 基础资料整理
- 使用注意事项


### 🌍 出海生活指南

整理：

- 国家与地区信息
- 长期居住参考
- 数字游民相关资料

```powershell

```

```
http://localhost:4173
```
