### 按资金门槛分类

| 资金要求 | 国家 | 路径 | 拿身份时间 | 核心门槛 |
|---------|------|------|-----------|---------|
| **几乎零成本** | 🇨🇦 加拿大 | EE 快速通道 / 省提名 | 6-12 个月 | 雅思 + 学历 + 工作经验 |
| **低成本** | 🇪🇸 西班牙 | 非盈利签证 | 5 年永居 | 80 万 RMB 存款 + 被动收入 |
| **中等成本** | 🇵🇹 葡萄牙 | D7 签证 | 5 年永居 | 3 万欧元 + 被动收入 |
| **中等成本** | 🇵🇦 巴拿马 | 退休年金签证 | 直接永居 | 25 万美元 + 月利息 850 美元 |

## 🇰🇷 韩国存款移民

> 韩国存款移民（公益事业投资移民）是目前东亚地区门槛最低的直接永居方案之一。存入指定金额到韩国银行，即可直接获得 F2 居留签证，5 年后转 F5 永居。

### 两种方案

| 方案 | 存款金额 | 身份 | 说明 |
|------|---------|------|------|
| **标准方案** | 15 亿韩元（约 800 万人民币） | F2 居留 → 5 年后 F5 永居 | 可自由支取利息 |
| **无息方案** | 30 亿韩元（约 1,600 万人民币） | 直接 F5 永居 | 无息存款，返还本金 |

> 1 亿韩元 ≈ 53 万人民币（2025年）。

### 基本要求

- 年龄：无限制
- 语言：无要求
- 居住：无强制居住要求（但转永居需在韩国有居住记录）
- 存款：需维持存款状态（F2 期间不可取出，F5 后可取出）
- 无犯罪记录

### 申请流程

1. 准备材料（护照、资金证明、无犯罪记录等）
2. 将资金汇入韩国指定银行（韩国产业银行）
3. 向韩国法务部提交申请
4. 获批后获得 F2 居留签证
5. 5 年后满足居住要求，申请 F5 永居

### 优势

- 无语言、无学历、无年龄要求
- 无需经营公司或找工作
- 存款可自由支取利息（标准方案）
- 距离中国近，往返方便
- 持 F2/F5 可自由进出韩国

### 劣势

- 资金门槛较高（800 万+人民币）
- 资金需冻结 5 年（标准方案）
- 韩国生活成本较高（尤其首尔）
- 韩国护照免签力一般（约 190 国，但不如香港/新加坡）

### 适合人群

- 高净值人士，有闲置资金
- 希望快速获得发达国家永居
- 需要在东亚地区有备用身份
- 计划子女在韩国接受教育

## 🇹🇷 土耳其护照投资

> 土耳其是目前全球唯一可以通过银行存款直接拿护照的大国。存入 50 万美元，维持 3 年后可取回，即可获得土耳其护照。

### 投资要求

| 项目 | 要求 |
|------|------|
| 存款金额 | 50 万美元（或等值外币） |
| 存款期限 | 维持 3 年 |
| 身份 | 一步到位拿护照 |
| 语言 | 无要求 |
| 居住 | 无要求 |

### 其他投资方式（可选）

- 购买 40 万美元以上房产（维持 3 年）
- 购买 50 万美元政府债券（维持 3 年）
- 创造 50 个就业岗位

### 护照优势

- 免签/落地签 110+ 国家/地区
- 可申请美国 E2 投资者签证（非绿卡，但可长期居住）
- 可申请英国商务签证
- 护照可改名（部分用户看重这一点）

### 劣势与风险

- **土耳其护照身份增益有限**：免签国数量一般，不如香港/新加坡/欧盟护照
- 土耳其经济波动较大，货币贬值风险
- 土耳其非欧盟成员国，无法享受欧盟自由流动
- **可能让你失去原来的身份福利**

### 适合人群

- 急需第二护照（如某些商业需求）
- 计划申请美国 E2 签证
- 有土耳其商业或家庭联系

> ⚠️ **不建议普通数字游民选择**：土耳其护照的性价比和实用性不如香港身份或日本永居。除非有特殊需求，否则优先考虑其他路径。

## 🇪🇸 西班牙非盈利签证

> 西班牙非盈利居留签证（Non-Lucrative Visa）是欧洲最受欢迎的低成本居留方案之一。无需投资、无需创业，只需证明有足够的被动收入或存款即可。

### 基本要求

| 项目 | 要求 |
|------|------|
| 资金证明 | 至少 80 万人民币等值存款（或每年约 2.8 万欧元被动收入） |
| 被动收入 | 房租、股息、养老金、远程工作收入等（非西班牙来源） |
| 医疗保险 | 西班牙境内全覆盖的私立医疗保险 |
| 无犯罪记录 | 中国和西班牙双国无犯罪 |
| 居住要求 | 每年需居住至少 183 天（实际执行可能有弹性） |

### 签证期限

| 阶段 | 期限 | 说明 |
|------|------|------|
| 首次签证 | 1 年 | 在西班牙领事馆申请 |
| 第一次续签 | 2 年 | 在西班牙境内申请 |
| 第二次续签 | 2 年 | 在西班牙境内申请 |
| 永居 | 5 年后 | 连续居住满 5 年可申请欧盟永居 |
| 护照 | 10 年后 | 连续居住满 10 年可申请西班牙护照 |

### 优势

- 欧盟申根区自由通行
- 无语言、无学历要求
- 资金门槛相对低
- 生活成本低于北欧/西欧
- 气候温暖，适合养老
- 可携带配偶和子女

### 劣势

- 不能在西班牙工作（受雇）
- 每年需住满 183 天（有移民监）
- 非盈利身份不可转盈利身份（需重新申请）
- 西班牙税务居民全球征税

### 适合人群

- 退休人士或有被动收入者
- 希望在欧盟生活但预算有限
- 数字游民（但需确保收入不被认定为西班牙工作收入）
- 陪读家庭

## 🇵🇹 葡萄牙 D7 签证

> 葡萄牙 D7 签证（被动收入签证）是西班牙非盈利签证的「姊妹版」，但门槛更低、政策更友好。5 年后可转欧盟永居或葡萄牙护照。

### 基本要求

| 项目 | 要求 |
|------|------|
| 被动收入 | 每月约 820 欧元（主申请人），配偶+50%，子女+30% |
| 存款证明 | 约 3 万欧元（覆盖一年生活费用） |
| 住所 | 葡萄牙租房或购房合同 |
| 医疗保险 | 葡萄牙全覆盖医疗保险 |
| 无犯罪记录 | 中国和葡萄牙双国无犯罪 |

### 被动收入类型

- 房租收入
- 股息、利息、理财收益
- 养老金
- 远程工作/自由职业收入（需注意税务安排）
- 版权费、专利费

### 签证期限

| 阶段 | 期限 | 说明 |
|------|------|------|
| 首次签证 | 4 个月入境签证 | 到葡萄牙后换居留卡 |
| 居留卡 | 1 年 | 在葡萄牙申请 |
| 续签 | 2 年 | 连续两次 |
| 永居 | 5 年后 | 可申请葡萄牙永居 |
| 护照 | 5 年后 | 可申请葡萄牙护照（欧盟护照） |

> 💡 葡萄牙护照 = 欧盟护照，可自由在欧盟 27 国生活、工作、学习。

### 优势

- 门槛低于西班牙非盈利（收入要求更低）
- 5 年后可拿欧盟护照（全球免签 180+ 国）
- 生活成本低于西欧其他国家
- 气候温暖，英语普及度较高
- 友好的税务政策（NHR 非惯常居民税务优惠）

### 劣势

- 需每年住满一定时间（有移民监）
- 不能在葡萄牙受雇（可远程工作）
- 行政手续较慢（葡萄牙 bureaucracy）
- 收入需持续证明

### 适合人群

- 有稳定被动收入（如房租、股息）
- 希望 5 年后拿欧盟护照
- 预算有限但想进入欧盟
- 数字游民/远程工作者

## 🇵🇦 巴拿马退休年金签证

> 巴拿马退休年金签证（Pensionado Visa）是拉美地区最成熟的退休居留方案，但其实对年龄没有限制，只要有稳定的年金/被动收入即可申请。

### 基本要求

| 项目 | 要求 |
|------|------|
| 年金/被动收入 | 每月至少 1,000 美元（如夫妻申请，配偶额外 +250 美元） |
| 存款 | 约 25 万美元（或等值资产证明） |
| 无犯罪记录 | 需认证 |
| 健康证明 | 巴拿马医生出具的健康证明 |

> 注：X @LuBtc888 提到的是「每月有 850 美元利息 + 25 万美元存款」，这与标准退休年金签证略有不同，可能是特定投资方案。以下为标准签证说明。

### 签证优势

| 项目 | 说明 |
|------|------|
| 永居权 | 直接获得永久居留权 |
| 无移民监 | 无最低居住要求 |
| 税务优惠 | 巴拿马属地征税，海外收入免税 |
| 美元经济 | 巴拿马使用美元，无汇率风险 |
| 生活成本 | 较低，尤其退休社区 |
| 家庭携带 | 配偶和子女可随行 |

### 退休福利（Pensionado 专属）

- 国际航班 25% 折扣
- 公共交通 30% 折扣
- 酒店 25-50% 折扣
- 电影院、剧院、体育赛事 25-50% 折扣
- 医疗 10-25% 折扣
- 公用事业账单 25% 折扣

### 劣势

- 巴拿马非发达国家，基础设施有限
- 英语普及度不如预期（西班牙语为主）
- 医疗水平中等（私立较好但费用不低）
- 治安不如东亚/西欧

### 适合人群

- 退休人士或有稳定年金收入
- 希望在低税国家有备用身份
- 预算有限，追求性价比
- 需要在美洲地区有落脚点

## 🇺🇸 美国 EB1A / NIW 人才移民

> 美国 EB1A（杰出人才）和 NIW（国家利益豁免）是低成本获取美国绿卡的路径，不需要雇主担保，仅靠材料申请即可。

### EB1A 杰出人才

#### 适合人群

- 科学家、研究员、教授
- 企业家、高管
- 艺术家、运动员
- 工程师、程序员（需有突出成就）

#### 十项标准（满足至少 3 项）

| 标准 | 说明 |
|------|------|
| 国家级奖项 | 获得过国家级或国际级奖项 |
| 协会会员 | 加入需要杰出成就的专业协会 |
| 媒体报道 | 专业媒体或主流媒体报道过你的成就 |
| 评审经历 | 担任过同行评审或评委 |
| 原创贡献 | 在领域内有原创性重要贡献 |
| 学术论文 | 发表过专业论文或著作 |
| 展览展示 | 作品被展览或展示（艺术领域） |
| 重要职务 | 在知名机构担任领导或关键职务 |
| 高薪 | 薪资远高于同行水平 |
| 商业成功 | 艺术作品取得商业成功 |

#### 费用与时间

| 项目 | 说明 |
|------|------|
| 律师费 | 约 1-3 万美元 |
| 官方费用 | 约 5,000 美元 |
| 总成本 | 约 1.5-3.5 万美元 |
| 排期 | 中国大陆出生者有排期（约 2-3 年） |
| 总时间 | 约 2-4 年 |

### NIW 国家利益豁免

#### 适合人群

- 高学历人才（硕士以上，或本科+5 年经验）
- 在科学、技术、工程、数学（STEM）等领域
- 工作对美国国家利益有重大贡献

#### 基本要求（满足以下任一）

| 路径 | 要求 |
|------|------|
| **高等学历** | 硕士以上，或本科+5 年相关工作经验 |
| **特殊能力** | 在科学、艺术或商业领域有杰出能力 |

#### 三要素测试

1. 你从事的工作对美国有**实质性价值**和**全国重要性**
2. 你处于**有利位置**来推进该工作
3. 豁免劳工证要求对美国**利大于弊**

#### 费用与时间

| 项目 | 说明 |
|------|------|
| 律师费 | 约 5,000-1.5 万美元 |
| 官方费用 | 约 5,000 美元 |
| 总成本 | 约 1-2 万美元 |
| 排期 | 中国大陆出生者有排期 |
| 总时间 | 约 3-5 年 |

### 优势与劣势

| 优势 | 劣势 |
|------|------|
| 无需雇主担保 | 材料要求高，需充分证明 |
| 无需投资 | 中国大陆出生者有排期 |
| 一步到位绿卡 | 竞争激烈，审核严格 |
| 全家可随行 | 需持续在领域内有贡献 |
| 绿卡永久有效 | 不是美国公民，无投票权 |

### 适合人群

- 学术研究者（有论文、引用、项目）
- 科技从业者（有专利、开源项目、行业影响力）
- 企业家（有成功创业经历）
- 有国际奖项或媒体报道

> 💡 美国绿卡持有者可享受大部分公民福利（教育、医疗、工作），但无投票权。5 年后可申请入籍。绿卡需每 10 年更新一次，且需证明与美国有持续联系。

## 🇨🇦 加拿大技术移民

> 加拿大是全球对技术人才最友好的移民国家之一，联邦快速通道（EE）和省提名（PNP）是两条主流路径。2025 年 EE 分数有望下降，是申请窗口期。

### 联邦快速通道（Express Entry）

#### 三大类别

| 类别 | 适合人群 | 核心要求 |
|------|---------|---------|
| **FSW（联邦技术工人）** | 海外技术工人 | 67 分打分 + 雅思 + 学历 + 工作经验 |
| **FST（联邦技术贸易）** | 技术工人（电工、焊工等） | 加拿大认证 + 雇主 offer |
| **CEC（加拿大经验类）** | 有加拿大工作/留学经验 | 加拿大 1 年工作经验 + 雅思 |

#### EE 打分系统（CRS）

| 评分项 | 最高分值 | 说明 |
|--------|---------|------|
| 年龄 | 20-29 岁 110 分 | 30 岁后分数递减 |
| 学历 | 博士 150 / 硕士 135 | 需加拿大或认证学历 |
| 语言能力 | 雅思 4 个 9 得 136 分 | 英语 + 法语可双加分 |
| 加拿大工作经验 | 5 年+ 80 分 | 加拿大境内工作经验 |
| 海外工作经验 | 3 年+ 50 分 | 与学历结合可双加分 |
| 配偶加分 | 最高 40 分 | 配偶学历/语言/工作经验 |
| 省提名 | +600 分 | 直接保证获邀 |
| 加拿大亲属 | +15 分 | 有加拿大公民/PR 亲属 |
| 法语能力 | +50 分 | 法语 NCLC 7 以上 |

#### 2025 年趋势

- 2025 年开春 EE 分数预计下降，是申请窗口期
- 法语能力（B2 以上）可大幅加分，甚至单独抽选
- 定向邀请（STEM、医疗、技工）分数较低

### 省提名项目（PNP）

> 省提名是加拿大各省根据自身需求选拔移民的机制。获得省提名后可在 EE 中加 600 分，几乎保证获邀。

#### 热门省份

| 省份 | 特点 | 适合人群 |
|------|------|---------|
| **安大略省（Ontario）** | 经济大省，多伦多是 tech 中心 | IT、金融、工程师 |

## 常见问题

### Q: 韩国存款移民和土耳其存款护照，选哪个？
A: **韩国更适合大多数中国用户**：距离近、文化相近、无语言要求、存款利息可自由支取。土耳其护照虽然门槛低，但护照增益有限，且土耳其经济不稳定。除非有特殊需求（如美国 E2），否则优先韩国。

### Q: 西班牙非盈利和葡萄牙 D7，选哪个？
A: **葡萄牙 D7 更友好**：门槛更低（每月收入要求更低）、5 年后可拿欧盟护照（西班牙需 10 年）、NHR 税务优惠更优。西班牙非盈利的优势是首次签证门槛低（只需存款）。如果追求欧盟护照，选葡萄牙。

### Q: EB1A 和 NIW 哪个更容易？
A: **NIW 门槛更低**：不需要国际级奖项，只需高学历 + 对美国国家利益的贡献。EB1A 要求更高（需满足 3 项杰出标准），但排期更快（EB1 类别优于 EB2）。如果材料不够硬，建议先申请 NIW。

### Q: 加拿大 EE 自己申请还是找中介？
A: **如果 CRS 分数够高（470+），完全可以 DIY**。EE 系统非常透明，官网有完整指引。如果分数不够高，可考虑：1) 提升雅思成绩；2) 学习法语；3) 找雇主走省提名；4) 咨询持牌移民顾问（RCIC）。

### Q: 以上所有路径都需要放弃中国国籍吗？
A: **大部分不需要**。韩国永居、西班牙非盈利、葡萄牙 D7、加拿大 PR、美国绿卡都是居留权，不影响中国国籍。只有入籍（拿护照）才需要放弃中国国籍。土耳其护照是直接拿护照，需要放弃中国国籍。**建议拿绿卡不入籍，保留双重福利。**

### Q: 这些路径有移民监吗？
A: | 国家 | 移民监 | 说明 |
|------|--------|------|
| 韩国 | 无强制（但转永居需有居住记录） | 较灵活 |
| 西班牙 | 每年 183 天 | 较严格 |
| 葡萄牙 | 每年 7 天（NHR 期间） | 较宽松 |
| 巴拿马 | 无 | 最灵活 |
| 美国 | 每 6 个月需入境一次 | 需保持联系 |
| 加拿大 | 每 5 年住满 2 年 | 标准移民监 |

### Q: 哪条路径性价比最高？
A: **取决于你的条件**：
- 有学历+英语好：加拿大 EE（几乎零成本）
- 有 800 万闲置资金：韩国存款移民（直接永居，无要求）
- 有被动收入：葡萄牙 D7（5 年拿欧盟护照）
- 行业有影响力：美国 EB1A/NIW（大国绿卡）
- 百强大学毕业生：香港高才通（最快最便宜）
