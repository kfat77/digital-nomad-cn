## 🇹🇭 泰国

### 泰国精英签证（Thailand Privilege）

> 前身为「Thailand Elite Visa」，2024 年重新品牌化为「Thailand Privilege」。这是一种付费会员制的长期居留方案，不是投资或移民签证。

#### 会员等级（2026年）

| 等级 | 有效期 | 会员费 | 特点 |
|------|--------|--------|------|
| Bronze | 5 年 | 约 90 万泰铢 | 基础方案，2026年9月30日前可申请 |
| Gold | 5 年 | 约 150 万泰铢 | 可升级至10年 |
| Platinum | 10 年 | 约 250 万泰铢 | 可添加家属 |
| Diamond | 15 年 | 约 350 万泰铢 | 高端权益 |
| Reserve | 20 年 | 约 500 万泰铢 | 邀请制，最高端 |

> 1 泰铢 ≈ 0.22 人民币（2025年）。90 万泰铢 ≈ 18 万人民币。

#### 包含权益

- 多次入境长期签证（有效期内自由出入）
- 每次入境可停留 1 年（可续期，无需离境）
- 机场 VIP 快速通关
- 政府礼宾服务（移民手续协助）
- 合作伙伴权益（酒店、医院、高尔夫球场等）
- 积分系统（可兑换服务）

#### 优势与劣势

| 优势 | 劣势 |
|------|------|
| 无年龄限制 | 无工作权利 |
| 无收入/资产要求 | 无税务优惠 |
| 无需每年续签 | 不是移民签证，无永居路径 |
| 流程简单，审核快 | 纯消费性质，无投资回报 |
| 适合长期度假/退休 | 政策可能随时调整 |

### 泰国 LTR 签证（长期居民签证）

> LTR Visa 是泰国为吸引高净值人才、退休人士、远程工作者和专业人士推出的 10 年签证（5+5 年）。

#### 四大类别

| 类别 | 目标人群 | 核心要求 |
|------|---------|---------|
| **全球富裕公民** | 高净值人士 | 资产 ≥100 万美元；年收入 ≥8 万美元 |
| **富裕退休人士** | 50 岁以上退休者 | 年收入 ≥8 万美元；或 ≥4 万+泰国投资 |
| **远程工作者** | 数字游民 | 为海外公司工作；年收入 ≥8 万美元 |
| **高技能专业人士** | 专业人才 | 为泰国公司/海外公司工作；年收入 ≥8 万 |

#### 通用要求

- 健康保险 ≥50,000 美元保额
- 无犯罪记录
- 护照有效期充足

#### LTR vs 精英签证

| 维度 | LTR | 精英签证 |
|------|-----|---------|
| 费用 | 较低（约 5 万泰铢申请费） | 较高（90 万+泰铢） |
| 工作权 | 部分类别有 | 无 |
| 税务优惠 | 有（境外收入免税） | 无 |
| 审核 | 较严格（需证明收入/资产） | 较宽松（付费即可） |
| 期限 | 10 年（5+5） | 5-20 年 |

### 泰国生活概要

| 项目 | 说明 |
|------|------|
| 生活成本 | 曼谷较高，清迈/普吉较低，整体低于国内一线城市 |
| 医疗 | 私立医院水平高，费用适中 |
| 气候 | 热带，全年炎热，3-5 月最热 |
| 语言 | 泰语为主，旅游区英语可用 |
| 数字游民 | 清迈、曼谷、普吉数字游民社区活跃 |

## 🇲🇾 马来西亚

### 马来西亚第二家园计划（MM2H）

> MM2H 是亚洲最知名的长期居留计划之一，2024 年重新改革，引入三档分级制度。

#### 三档对比（2024-2025）

| 等级 | 固定存款 | 房产要求 | 签证期限 | 工作权 | 年龄 |
|------|---------|---------|---------|--------|------|
| **Silver** | USD 15 万 | 60 万马币 | 5 年 | 无 | ≥30 岁 |
| **Gold** | USD 50 万 | 100 万马币 | 15 年 | 无 | ≥30 岁 |
| **Platinum** | USD 100 万 | 200 万马币 | 20 年 | 有 | ≥30 岁 |

> 1 马币 ≈ 1.6 人民币。60 万马币 ≈ 96 万人民币（房产）。

#### 通用要求

- 每年在马来西亚居住 ≥60 天（2024年后从 90 天降至 60 天）
- 固定存款需维持（部分可提取用于购房、教育、医疗）
- 不可出售房产（10 年内），除非替换更高价值房产
- 配偶、未婚子女（≤34 岁）、父母/岳父母可随行

#### 特殊经济区方案（SEZ/SFZ）

- 针对森林城等特定区域
- 21-49 岁：USD 6.5 万存款
- 50 岁以上：USD 3.2 万存款
- 需购买指定开发商房产
- 10 年签证

#### 优势与劣势

| 优势 | 劣势 |
|------|------|
| 华语环境好，华人占比 23% | 气候炎热潮湿 |
| 生活成本较低 | 无直接永居/公民路径 |
| 教育资源丰富（国际课程） | 工作权受限（Platinum 除外） |
| 境外收入免税 | 部分区域治安需注意 |
| 医疗水平高，费用低 | 政策不稳定（多次调整） |

### 马来西亚 Premium Visa Programme（PVIP）

| 项目 | 要求 |
|------|------|
| 月收入 | ≥40,000 马币（约 9,700 美元）离岸收入 |
| 固定存款 | 100 万马币（约 24 万美元） |
| 参与费 | 20 万马币 |
| 签证 | 20 年（可续签） |
| 最低居住 | 无要求 |
| 工作权 | 可经商，不可受雇 |

> PVIP 适合高净值人士，门槛高于 MM2H，但更灵活。

### 马来西亚生活概要

| 项目 | 说明 |
|------|------|
| 生活成本 | 吉隆坡中等，槟城/新山较低 |
| 医疗 | 私立医院水平高，费用极低 |
| 气候 | 热带雨林，全年炎热多雨 |
| 语言 | 华语、马来语、英语通用 |
| 数字游民 | 吉隆坡、槟城社区活跃 |

## 🇸🇬 新加坡

### 全球投资者计划（GIP）

> 新加坡最顶级的投资者居留计划，直接申请永久居民（PR）。

| 方案 | 投资要求 | 说明 |
|------|---------|------|
| 方案 A | 1,000 万新元设立新企业 | 或投资现有企业 |
| 方案 B | 2,500 万新元投入 GIP 精选基金 | 需符合资格基金 |
| 方案 C | 设立家族办公室（管理资产 ≥2 亿新元） | 针对超高净值 |

> 1 新元 ≈ 5.4 人民币。1,000 万新元 ≈ 5,400 万人民币。

#### 申请要求

- 拥有丰富商业经验
- 有成功的企业运营记录
- 提交详细的商业计划或投资方案
- 通过面试

### 新加坡 EP/SP 工作准证

> 更现实的路径：先拿工作准证，2 年后申请 PR。

| 准证 | 要求 | 有效期 |
|------|------|--------|
| **EP（就业准证）** | 月薪 ≥5,000 新元（金融业更高）；学历/经验 | 1-2 年 |
| **SP（S准证）** | 月薪 ≥3,000 新元；技术岗位 | 1-2 年 |
| **COMPASS 框架** | 2023 年起 EP 需通过积分评估（薪资、学历、多样性等） | — |

#### 拿 PR 路径

```
EP/SP 工作准证 → 持续工作 1-2 年 → 申请 PR → 审核通过 → 新加坡 PR
```

> 新加坡 PR 审批较严格，不保证获批。拥有本地学历、家庭在新加坡、纳税记录良好会增加成功率。

### 新加坡生活概要

| 项目 | 说明 |
|------|------|
| 生活成本 | 全球最高之一（房租、餐饮、交通） |
| 医疗 | 顶尖水平，但费用高（需保险） |
| 气候 | 热带，全年炎热潮湿 |
| 语言 | 英语为主，华语、马来语通用 |
| 数字游民 | 金融科技中心，创业社区活跃 |

## 常见问题

### Q: 东南亚居留签证能拿永居或护照吗？
A: **绝大多数不能直接拿永居**。泰国精英签证、马来西亚 MM2H、印尼黄金签证都是长期居留，不是移民。新加坡 GIP 可直接拿 PR，但门槛极高。新加坡 EP 工作满 1-2 年可申请 PR，但审批不保证。

### Q: 持有东南亚居留签证可以工作吗？
A: 泰国精英签证和马来西亚 MM2H（Silver/Gold）**不可工作**。马来西亚 Platinum 可工作。泰国 LTR 部分类别有工作权。新加坡 EP/SP 本身就是工作准证。

### Q: 东南亚居留签证和数字游民签证有什么区别？
A: 数字游民签证（如泰国 DTV）是短期（1 年），面向远程工作者。居留签证（如精英签证、MM2H）是长期（5-20 年），面向退休、投资或定居。部分国家两者有重叠。

### Q: 马来西亚 MM2H 的政策稳定吗？
A: **不太稳定**。2021 年曾暂停，2023 年大幅收紧，2024 年又重新放宽。建议申请前确认最新政策。这是东南亚国家居留计划的共同特点——政策可能频繁调整。

### Q: 新加坡 PR 好拿吗？
A: **不容易**。新加坡 PR 审批不透明，竞争非常激烈。拥有本地学历、在新加坡组建家庭、长期稳定纳税、行业稀缺性都有帮助。但没有任何保证。

### Q: 东南亚哪个国家最适合数字游民？
A: 综合生活成本、社区、签证便利性：**泰国（清迈/曼谷）和马来西亚（吉隆坡/槟城）** 是首选。巴厘岛适合短期停留，但签证不太方便。新加坡适合短期活动和高薪工作，但生活成本极高。

## 📝 东南亚居留建议

1. ✅ 明确目标：是退休？远程工作？还是长期定居？
2. ✅ 评估预算：一次性投入 vs 每年维护成本
3. ✅ 了解限制：是否可以工作？是否有居住要求？
4. ✅ 关注政策：东南亚国家政策变化频繁，务必核实最新信息
5. ✅ 实地考察：先以旅游签去住 1-3 个月，再决定是否申请
6. ✅ 购买保险：医疗险是必备，尤其是东南亚私立医疗费用不低
