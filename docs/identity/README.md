# 身份模块

> 第二身份是数字游民最核心的底层资产。一个合法合规的第二身份，意味着全球通行自由、资产安全隔离、教育资源获取和税务优化空间。
>
> **💡 核心建议：选对国家 + 拿绿卡不入籍 = 双重福利 + 出入境便利。** 参考来源：X @LuBtc888

## 🗺️ 全球移民路径速览（按门槛分类）

> 以下分类综合了 X @LuBtc888 的移民门槛分析和各国官方政策，帮助你快速定位适合自己的方案。

### 1. 有充足存款，但不想花钱（存款即移民）

| 国家 | 门槛 | 拿身份 | 资金冻结 | 特点 |
|------|------|--------|---------|------|
| 🇰🇷 **韩国** | 存入 15 亿韩元（无息 30 亿） | 直接永居绿卡 | 需维持存款 | 东亚，无语言要求，距离近 |
| 🇹🇷 **土耳其** | 存入 50 万美元 | 一步到位护照 | 需维持 3 年 | 护照免签 110+ 国，但小国护照增益有限 |
| 🇪🇸 **西班牙** | 非盈利签证，80 万 RMB 存款 + 非工资收入证明 | 1 年签 → 续签 → 5 年永居 | 需证明收入 | 欧盟申根，生活成本适中 |
| 🇵🇹 **葡萄牙** | D7 签证，存入 3 万欧元 + 被动收入 | 1 年签 → 续签 → 5 年永居 | 需证明收入 | 欧盟申根，5 年后可转护照 |
| 🇵🇦 **巴拿马** | 退休年金签证，25 万美元 + 月利息 850 美元 | 直接永居 | 需维持投资 | 拉美跳板，税务友好 |

> ⚠️ **提示：别考虑小国护照（加勒比岛国等）**，没有任何身份增益，还会让你失去原来的身份福利。建议拿绿卡不入籍，保留双重福利。

### 2. 低成本去发达国家（没钱但能力强）

| 国家 | 路径 | 核心要求 | 成本 | 时间 |

## ⚠️ 重要提醒

1. **身份规划没有「最优解」**，只有「最适合你的方案」
2. 政策随时变化，请以各国/地区官方最新信息为准
3. 建议咨询持牌移民律师或顾问，本指南仅供参考
4. 所有移民申请均需真实材料，虚假材料会导致拒签并留下不良记录
5. 中国公民申请海外身份时，需遵守中国相关法律法规
6. **建议拿绿卡不入籍**，保留中国国籍福利 + 享受出入境便利
