# 身份模块

> 第二身份是数字游民最核心的底层资产。一个合法合规的第二身份，意味着全球通行自由、资产安全隔离、教育资源获取和税务优化空间。

## 🌍 为什么需要规划第二身份？

| 需求 | 限制 | 第二身份解决方案 |
|------|------|----------------|
| 全球免签/落地签 | 中国护照约 80+ 免签国 | 香港特区护照 170+ 免签 |
| 子女国际教育 | 外籍学校/海外留学门槛高 | 港澳台身份享受国际生政策 |
| 资产全球配置 | 外汇管制每年 5 万美元 | 境外身份账户无额度限制 |
| 税务优化 | 全球征税（CRS） | 部分司法管辖区税收优惠 |
| 备用安全港 | 单一国籍风险 | 多身份避险 |
| 长期海外居住 | 签证限制 | 永久居留/公民身份 |

## 🗺️ 全球移民路径速览

### 按资金门槛分类

| 资金要求 | 国家/地区 | 路径 |
|---------|----------|------|
| 几乎零成本 | 加拿大、新西兰 | 技术移民、雇主担保 |
| 低成本 | 香港、日本、西班牙 | 人才计划、工签转永居 |
| 中等成本 | 马来西亚、泰国 | 存款/投资居留 |
| 高成本 | 新加坡、澳大利亚、美国 | 投资者签证、人才移民 |
| 极高成本 | 土耳其、马耳他、加勒比岛国 | 护照投资 |

### 按时间周期分类

| 拿身份时间 | 国家/地区 | 路径 |
|-----------|----------|------|
| 最快（1-2年） | 香港、日本 | 高才通、高度人才 |
| 中等（3-5年） | 澳大利亚、加拿大、新西兰 | 技术/雇主担保 |
| 较长（5-10年） | 日本、欧洲各国 | 普通工签/经营签转永居 |
| 最长（10年+） | 日本普通路径 | 普通工签/经营签 |

## ⚠️ 重要提醒

1. **身份规划没有「最优解」**，只有「最适合你的方案」
2. 政策随时变化，请以各国/地区官方最新信息为准
3. 建议咨询持牌移民律师或顾问，本指南仅供参考
4. 所有移民申请均需真实材料，虚假材料会导致拒签并留下不良记录
5. 中国公民申请海外身份时，需遵守中国相关法律法规
