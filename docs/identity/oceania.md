## 🇦🇺 澳大利亚

### 2025-2026 年政策核心变化

| 变化 | 详情 |
|------|------|
| **总配额** | 18.5 万/年，技术移民占 71%（约 13.2 万） |
| **职业清单** | CSOL 核心技能清单扩展至 456 个职业 |
| **新签证** | NIV 国家创新签证取代 GTI；SID 取代 482 |
| **偏远地区** | 州担保（190/491）和偏远雇主担保优先审理 |
| **语言要求** | 部分签证提升（如学生签雅思 6 分） |
| **薪资门槛** | 核心技能签证年薪门槛 7.6 万澳元 |

### 技术移民（189/190/491）

#### 189 独立技术移民

| 项目 | 要求 |
|------|------|
| 年龄 | ≤45 岁 |
| 职业 | 在 MLTSSL 清单上 |
| 打分 | EOI ≥65 分（竞争激烈，热门职业需 90+） |
| 英语 | 雅思 4 个 6 或同等 |
| 优势 | 一步到位 PR，无需担保 |
| 劣势 | 竞争激烈，高分职业（IT、工程、会计）需 90+ 分 |

#### 190 州担保技术移民

| 项目 | 要求 |
|------|------|
| 州担保 | 需获得州/领地政府提名 |
| 职业 | 在州政府需求清单上 |
| 打分 | EOI 获担保后 +5 分 |
| 优势 | 分数要求相对低，各州需求不同 |
| 注意 | 部分州要求承诺在该州居住 2 年 |

#### 491 偏远地区技术移民

| 项目 | 要求 |
|------|------|
| 担保 | 州政府或偏远地区亲属担保 |
| 居住 | 在偏远地区居住和工作 3 年 |
| 转 PR | 满 3 年后转 191 PR（无需再担保） |
| 优势 | 分数要求低，偏远地区范围已扩大 |
| 变化 | 2025 年后转永居居住期从 5 年缩短至 3 年 |

#### 打分系统（EOI）

| 评分项 | 最高分值 |
|--------|---------|
| 年龄 | 25-32 岁 30 分 |
| 英语 | 雅思 4 个 8 得 20 分 |
| 学历 | 博士 20 / 本科 15 |
| 澳洲工作经验 | 8 年+ 20 分 |
| 海外工作经验 | 8 年+ 15 分 |
| 配偶加分 | 配偶英语/技能/学历 5-10 分 |
| 州担保 | 190 加 5 / 491 加 15 |
| 偏远地区学习 | 澳洲偏远地区学习 5 分 |
| 职业年/CCL | 5 分 |

### 雇主担保移民

#### 186 雇主担保永居（ENS）

| 项目 | 要求 |
|------|------|
| 年龄 | ≤45 岁 |
| 经验 | 3 年相关工作经验 |
| 雇主 | 澳洲雇主提名真实职位 |
| 职业 | 在 CSOL 清单上（456 个职业） |
| 优势 | 直接 PR，无需打分 |

#### 482 SID 临时签证（取代旧 482）

| 项目 | 要求 |
|------|------|
| 经验 | 1 年相关工作经验（2025 年新政） |
| 职业 | 在 CSOL 清单上 |
| 雇主 | 澳洲雇主担保 |
| 转 PR | 为雇主工作 2 年后转 186 TRT |
| 优势 | 职业范围扩大（456 个），门槛降低 |

#### 494 偏远地区雇主担保

| 项目 | 要求 |
|------|------|
| 区域 | 非大都市区域 |
| 转 PR | 工作 3 年后转 191 PR |
| 优势 | 可带家属，福利齐全 |

### NIV 国家创新签证（取代 GTI）

> 2025 年推出的新签证，面向全球顶尖人才，对标美国 EB1A/NIW。

#### 目标人群

- 国际学术研究者（科研成果显著）
- 创业家（成功运营创新企业）
- 投资人（创新企业投资经验）
- 体育/创意人才（国际影响力）

#### 五大重点领域（新州政府提名）

- 资源、农业食品、能源
- 健康、国防、循环经济
- 数字技术、基础设施
- 金融服务、教育

#### 特点

- 直接 PR，无需雇主担保
- 需先提交 EOI，获政府提名后申请
- 年薪门槛或优秀博士毕业生

### 澳大利亚生活概要

| 项目 | 说明 |
|------|------|
| 生活成本 | 悉尼/墨尔本高，其他城市中等 |
| 医疗 | Medicare 全民医保，免费公立医院 |
| 教育 | 公立学校免费，大学对 PR 学费远低于国际生 |
| 气候 | 多样，南部四季分明，北部热带 |
| 语言 | 英语 |
| 工作 | 职场文化相对轻松，WLB 好 |

## 🇳🇿 新西兰

### 2025-2026 年政策核心变化

| 变化 | 详情 |
|------|------|
| **技术移民** | 2026 年 8 月起优化，本地经验更受青睐 |
| **雇主担保** | AEWV 签证岗位扩容 80+，取消强制培训模块 |
| **学生签证** | 合法工作时长从 20 小时/周提升至 25 小时 |
| **投资移民** | 门槛调整，资本与创新获更多政策支持 |
| **核心方向** | 管控总量 → 提升质量 → 精准对接经济需求 |

### 技术移民（六分制）

> 2023 年取代旧 180 分制，改为六分制，更加简洁。

| 评分来源 | 分值 | 说明 |
|---------|------|------|
| 注册职业 | 3/4/6 分 | 根据培训时长 |
| 学历 | 3/4/6 分 | 本科 3 / 硕士 5 / 博士 6 |
| 收入 | 3/4/6 分 | 1.5/2/3 倍中位数 |
| 新西兰工作 | 每年 +1 分 | 最多 3 分 |

**需总分 ≥6 分，且必须有新西兰雇主 offer。**

#### 基本要求

- 年龄 ≤55 岁
- 英语雅思 6.5 或同等
- 本科学历以上
- 专业满足新西兰需求
- 薪资达到当地中位数
- 健康和品格要求

### 雇主担保签证（AEWV）

> 2025 年 11 月起重大改革，担保岗位大幅增加，审批提速。

#### 变化

- 担保岗位覆盖 IT、建筑工程、健康护理、教育等 80+ 新增岗位
- 取消雇主强制培训模块，减轻合规负担
- 审核速度全面提升

#### 路径

```
获得雇主 offer → 申请 AEWV → 工作 2-3 年 → 申请技术移民或直接转 PR
```

### 新西兰投资移民

| 类别 | 要求 | 说明 |
|------|------|------|
| 积极投资者 Plus | 投资 500 万-1,500 万新西兰元 | 需维持投资 3-4 年 |
| 创新/创业 | 创新商业计划 | 需经商业移民评估 |

### 新西兰永久回头签（Permanent Resident Visa）

> **全球唯一**：拿到永久居民签证后，可无限期离开新西兰，回国居住无时间限制，永远保留回头权利。

#### 与澳大利亚 PR 的对比

| 维度 | 新西兰 | 澳大利亚 |
|------|--------|---------|
| 居住要求 | 拿到永久回头签后无要求 | 每 5 年需住满 2 年（居民往返签证） |
| 回头权利 | 永久 | 有条件（需满足居住要求） |
| 灵活性 | 极高 | 中等 |
| 适合 | 需要回国长期居住的申请人 | 计划在澳长期居住的申请人 |

### 新西兰生活概要

| 项目 | 说明 |
|------|------|
| 生活成本 | 奥克兰高，其他城市中等，整体低于澳洲大城市 |
| 医疗 | 公立医疗免费，但等待时间较长 |
| 教育 | 公立学校免费，大学对 PR 学费低 |
| 气候 | 温带海洋性，四季温和但多雨 |
| 语言 | 英语、毛利语 |
| 工作 | 职场文化轻松，但薪资天花板低于澳洲 |
| 自然 | 风景绝美，户外运动天堂 |

## 两国对比

### 核心差异

| 维度 | 澳大利亚 | 新西兰 |
|------|---------|--------|
| 人口 | 2,600 万 | 520 万 |
| 经济体量 | 大（G20 成员） | 小（农业/旅游为主） |
| 技术移民难度 | 高（竞争激烈） | 中等（需雇主 offer） |
| 永居灵活性 | 需满足居住要求 | 永久回头签（全球唯一） |
| 薪资水平 | 更高 | 较低 |
| 生活成本 | 更高 | 较低 |
| 气候 | 更多样（热带到温带） | 温带海洋性，多雨 |
| 大城市 | 悉尼、墨尔本、布里斯班 | 奥克兰、惠灵顿、基督城 |
| 就业机会 | 更多（IT、金融、矿业） | 较少（农业、旅游、教育） |
| 移民监 | 有 | 拿到回头签后无 |

### 推荐路径

| 人群 | 推荐 |
|------|------|

| 有澳洲雇主 offer | 澳大利亚 SID → 186 |
| 学术/研究人才 | 澳大利亚 NIV 或新西兰技术移民 |

## 常见问题

### Q: 澳大利亚和新西兰可以互相流动吗？
A: **有特定通道**。新西兰公民可在澳大利亚自由工作和居住（特殊类别签证 444），但需注意税务和福利差异。澳大利亚公民也可在新西兰自由居住和工作。但两国移民政策各自独立，拿到一国身份不自动获得另一国身份。

### Q: 技术移民的「职业清单」是什么？
A: 澳大利亚有 MLTSSL（中长期清单）、STSOL（短期清单）、CSOL（核心技能清单）。新西兰有长期技能短缺清单（LTSSL）。你的职业必须在清单上才能申请技术移民。IT、工程、医疗、教育、护理是目前最热门的短缺职业。

### Q: 没有海外学历可以申请吗？
A: **可以**。但需要先通过职业评估（Skills Assessment）。中国学历通常需要做认证（如 VETASSESS、ACS、EA 等机构的评估）。部分职业还需要额外的工作经验认证。

### Q: 年龄超过 45 岁还能移民吗？
A: 澳大利亚技术移民和大部分雇主担保要求 ≤45 岁。新西兰技术移民要求 ≤55 岁。如果超过年龄限制，可考虑：澳大利亚 186 雇主担保（某些豁免）、全球人才签证、或投资移民路径。

### Q: 英语要求高吗？
A: 澳大利亚技术移民通常要求雅思 4 个 6（或 PTE 同等）。新西兰要求雅思 6.5。部分职业（如护理、教师）要求更高。如果英语不够，可先考出成绩再申请，或在澳洲/新西兰读语言课程。

### Q: 拿到 PR 后需要住多久？
A: **澳大利亚**：PR 本身无期限，但居民往返签证（RRV）每 5 年需住满 2 年，否则需证明与澳洲有紧密联系。**新西兰**：拿到永久回头签后无居住要求，可无限期离开。

### Q: 可以带家属吗？
A: 可以。配偶和子女可随行。主申请人的技术移民/雇主担保通常包含家属。配偶如英语达标或从事技能职业，还可能为主申请人加分。

### Q: 大洋洲移民和数字游民生活冲突吗？
A: 不冲突。拿到 PR 后，你可以自由出入，同时保持全球数字游民生活方式。新西兰永久回头签尤其适合这种灵活生活方式。澳大利亚 PR 持有者也可在满足条件的情况下长期离境。

## 📝 大洋洲移民建议

### 推荐路径（按优先级）

1. **有澳洲/新西兰雇主 offer**：直接申请雇主担保（最快路径）
2. **高技能人才**：评估技术移民打分，189/190/491 或新西兰六分制
3. **留学生**：澳洲/新西兰高校 → 毕业生工签 → 积累本地经验 → 技术移民
4. **顶尖人才/投资者**：澳大利亚 NIV 或投资移民
5. **需要灵活回国**：优先考虑新西兰（永久回头签）

### 准备清单

- [ ] 评估职业是否在技能清单上
- [ ] 完成职业评估（如需要）
- [ ] 考出英语成绩（雅思/PTE）
- [ ] 计算 EOI 打分（澳洲）或六分制（新西兰）
- [ ] 研究目标州/城市（各州政策不同）
- [ ] 准备学历、工作经验、资金证明等材料
- [ ] 考虑先留学获取本地学历和工作经验
