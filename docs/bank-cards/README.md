# 银行卡模块

> 境外银行卡是中国数字游民最基础的基础设施之一。拥有一张境外银行卡，可以解决跨境消费、海外投资、资金自由兑换等核心需求。

## 🏦 为什么要办境外银行卡？

| 痛点 | 国内卡 | 境外银行卡 |
|------|--------|-----------|
| 境外消费手续费 | 1%-2%货币转换费 + 可能额外手续费 | 通常无手续费或极低 |
| 跨境汇款 | 额度受限、手续繁琐 | 便捷、低成本 |
| 绑定海外服务 | 经常被拒 | 正常绑定 |
| 投资港美股 | 无法直接出入金 | 直接关联券商账户 |
| 多币种管理 | 需单独开通 | 多币种账户天然支持 |

## 🌍 热门开户地区

| 地区 | 优势 | 代表银行 | 开户难度 |
|------|------|---------|---------|
| 🇭🇰 香港 | 距离近、政策相对宽松、中文化好 | 汇丰、中银、众安 | ⭐⭐ 较易 |
| 🇸🇬 新加坡 | 金融稳定、隐私保护强 | 华侨、星展 | ⭐⭐⭐ 中等 |
| 🇺🇸 美国 | 全球通用性强 | Chase、BoA | ⭐⭐⭐⭐ 较难 |
| 🇬🇧 英国 | 欧洲通用 | HSBC UK | ⭐⭐⭐ 中等 |
