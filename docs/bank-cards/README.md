## 四卡策略概览

| 卡片 | 银行 | 用途 | 开户难度 |
|------|------|------|----------|
| **香港汇丰** | HSBC HK | 全球转账、多币种 | 中等（需赴港或见证开户） |
| **香港中银** | BOC HK | 港币结算、内地互通 | 中等 |
