## Wise 是什么

Wise 是一家持牌金融科技公司，总部位于英国伦敦，受 FCA（英国金融行为监管局）监管。它提供：

| 功能 | 说明 |
|------|------|
| **多币种账户** | 持有 50+ 种货币 |
| **本地银行账户** | 获得 40+ 国家的本地银行账户（如美国 ACH、英国 IBAN、新加坡 PayNow 等） |
| **跨境转账** | 以真实汇率（无隐藏加价）进行国际汇款 |
| **Wise 借记卡** | 实体卡/虚拟卡，全球消费 |
| **合规出入金** | 支持连接券商、加密货币平台 |

### ❌ 它不能做什么

| 不能 | 说明 |
|------|------|
| 不是传统银行 | 无存款保险（如 FDIC/SIPC），但资金由合作银行托管 |
| 不能存现金 | 只能电子转账充值 |
| 不提供贷款/信用卡 | 纯支付/转账工具 |

> 💡 一句话：**Wise 不是银行，但功能上等同于一个全球多币种账户 + 跨境转账神器。**

## 注册流程详解

**官方入口**：[wise.com](https://wise.com)

1. 访问 wise.com，点击 "Register"
2. 选择 "Personal account"（个人账户）
3. 填写邮箱，设置密码
4. 验证邮箱（点击邮件中的链接）
5. 填写个人信息（姓名拼音、出生日期、国籍 China）
6. 选择要开通的货币账户（建议先开通 USD、SGD、GBP）
7. **身份验证**：上传身份证正反面照片
8. 等待审核（通常几分钟到几小时）

> 💡 根据实测经验（X @0xJoveXu），大陆身份证注册 Wise 审核通常 5 分钟内通过。

### 中国区 Wise 实体卡恢复申请

> 2024-2025 年，中国区 Wise 一度暂停实体卡申请。目前已恢复，大陆/香港/台湾地区用户均可申请。

申请方式：
1. 登录 Wise App
2. 进入 "Cards" 页面
3. 点击 "Order a card"
4. 选择实体卡或虚拟卡
5. 填写邮寄地址（国内地址可邮寄）
6. 支付卡费，等待邮寄

## 功能使用

### 多币种账户

开通后，你会获得以下账户信息：

| 货币 | 账户类型 | 用途 |
|------|---------|------|
| USD | ACH 路由号码 | 美国转账、券商入金 |
| GBP | IBAN + Sort Code | 英国转账 |
| EUR | IBAN | 欧元区转账 |
| SGD | PayNow / FAST | 新加坡转账 |
| AUD | BSB + 账号 | 澳大利亚转账 |

> 💡 每个货币账户都有独立的银行账号，别人可以像转账给本地银行账户一样转账给你。

### Wise 转账

**向别人转账**：
1. 选择 "Send money"
2. 选择转出货币和金额
3. 选择转入货币
4. 填写收款人信息（银行账户或 Wise 账号）
5. 确认汇率和手续费
6. 完成转账

**收款**：
- 直接将你的 Wise 本地银行账户信息提供给付款方
- 支持银行转账、信用卡/借记卡、Apple Pay/Google Pay 等多种方式

## 出入金指南

### 向 Wise 入金

| 方式 | 说明 | 手续费 |
|------|------|--------|
| 国内银行转账 | 购汇后电汇到 Wise | 国内银行电汇费 + Wise 入账费 |
| 支付宝 | 部分场景支持 | 较低 |
| 其他 Wise 用户转账 | 即时到账 | 免费 |
| 工资/平台收益 | 直接打到 Wise 本地账户 | 通常免费 |

### 从 Wise 出金

| 方式 | 说明 | 手续费 |
|------|------|--------|
| 转回国内银行卡 | 结汇回国 | 较低 |
| 转到香港/新加坡银行 | 跨境转账 | 低 |
| 券商入金 | 盈透、嘉信等 | 通常免费或极低 |
| Wise 借记卡消费 | 直接刷卡 | 无额外手续费 |

### 连接券商

**Wise → 盈透证券（IBKR）**：
1. 在 IBKR Client Portal 获取 USD 入金账户信息
2. 在 Wise 中选择 "Send money" → 添加 IBKR 的美国银行账户
3. 填写 ACH 路由号和账号
4. 确认转账（通常 1-2 个工作日到账）

> 💡 Wise 的 USD 账户支持 ACH 转账，这是美国最常见的银行间转账方式，兼容几乎所有美股券商。

## 常见问题

### Q: Wise 安全吗？
A: Wise 持有英国 FCA 牌照，资金由合作银行托管，且实行客户资金隔离。但注意：**它不是银行，没有存款保险**。建议不要存放大额资金，作为支付/转账工具使用即可。

### Q: 大陆身份证会被拒吗？
A: 一般不会。Wise 对中国大陆用户开放注册，审核通常很快。但如果资料填写有误或触发风控，可能需要补充材料。

### Q:  Wise 能替代香港银行卡吗？
A: **部分替代**。Wise 适合跨境转账、多币种管理、连接券商。但香港银行卡在以下场景更有优势：港陆互转免费、本地消费、香港信用卡、部分香港本地服务。两者可以互补使用。

### Q: Wise 实体卡能在国内用吗？
A: 可以。Wise 实体卡是 Visa/Mastercard，可在国内支持外卡的 POS 机消费。但国内大部分商户不支持外卡，建议主要用作海外消费和线上支付。

### Q: 从 Wise 转钱回国内银行卡有限额吗？
A: 受中国外汇管制影响，个人年度便利化购汇/结汇额度为等值 5 万美元。超出部分需合规申报。

### Q: Wise 和 OCBC 选哪个？
A: | 维度 | Wise | OCBC |
|------|------|------|
| 开户难度 | 极低（身份证即可） | 低（需护照+身份证+NFC） |
| 实体卡 | 有（Visa/Mastercard） | 有（Visa/Mastercard） |
| 本地账户数量 | 40+ 国家 | 新加坡为主 |
| 转账费率 | 极低 | 新加坡内免费，跨境有费 |
| 适合 | 跨境收款/转账 | 新加坡本地业务 |

> 💡 **建议两者都开**：Wise 做全球转账和收款，OCBC 做新加坡本地业务和备用。
