## 为什么选择 OCBC

| 优势 | 说明 |
|------|------|
| 🆔 远程开户 | 无需去新加坡，手机 App 在线完成 |
| 🛡️ 资金安全 | 新加坡三大银行之一，MAS 监管 |
| 💱 多币种 | 支持 SGD、USD、CNY 等多种货币 |
| 💳 实体借记卡 | 可申请 Visa/Mastercard 实体卡 |
| 🌐 中文服务 | App 和客服支持中文 |
| 🔗 连接券商 | 可入金美股/港股券商 |

**适合人群**：
- 不想去香港/新加坡线下开户
- 需要新加坡银行账户作为海外资产配置
- 需要一张新加坡实体银行卡

## 开户准备

| 材料 | 说明 |
|------|------|
| 大陆身份证 | 正反面照片 |
| 护照 | 有效期内，需支持 NFC 读取 |
| 大陆手机号 | 接收验证码 |
| 邮箱 | 用于账户通知 |
| 手机支持 NFC | 用于读取护照芯片信息 |

> ⚠️ **重要**：开户过程需要通过 NFC 读取护照芯片信息，请确保手机支持 NFC 功能。

## 开户流程详解

**官方入口**：下载 **OCBC Digital** App（iOS/Android）

### 第一步：下载 App

1. 在 App Store / Google Play 搜索 "OCBC Digital"
2. 下载并安装
3. 打开 App，点击 "Open Account"

> 💡 如果国内 App Store 搜不到，可能需要切换到新加坡/香港 Apple ID。

### 第二步：填写个人信息

1. **选择账户类型**：Personal Account（个人账户）
2. **国籍**：China
3. **姓名**：拼音大写（与护照一致）
4. **出生日期**
5. **联系方式**：邮箱 + 大陆手机号（Country code: 86）
6. **居住地址**：如实填写
   - 如果邮寄地址与身份证地址不同，选择 "Use a different mailing address"
   - 说明原因（如：目前在泰国生活和居住）

### 第三步：扫描护照

1. 打开护照，找到带有条形码的页面
2. 用手机扫描护照条形码
3. **NFC 读取护照芯片**：
   - 将手机背面贴近护照封面（有芯片的位置）
   - 如果没反应，上下小幅度移动手机
   - 有反应后保持不动，等待读取完成

> 💡 这是最关键的一步。部分国产手机 NFC 灵敏度较低，可能需要多试几次。

### 第四步：补充信息

1. **就业信息**：公司名称、职务、个人年收入（如实填写）
2. **资金来源**：Salary / Wages / Savings
3. **账户用途**：General banking / Investment

### 第五步：设置账户

1. 设置用户名（登录网银用，记好别忘）
2. 设置 6 位数字密码（登录密码 + ATM 取款密码）
3. 开通面容 ID / 指纹登录

### 第六步：等待审核

- 部分用户秒通过（实测案例：X @JohnnyBi_X）
- 部分用户可能需要 1-3 个工作日
- 通过后收到邮件和短信通知

## 开户后设置

### 查看账户信息

1. 登录 OCBC Digital App
2. 点击 "Your Assets"（你的资产）
3. 向下滑动查看账号信息

### 获取银行信息（用于国际汇款）

| 信息 | 用途 |
|------|------|
| 银行名称 | OCBC Bank |
| SWIFT 代码 | OCBCSGSG |
| 银行地址 | 65 Chulia Street, OCBC Centre, Singapore 049513 |
| 账号 | 你的 OCBC 账号 |

> 💡 如需从其他银行汇款到 OCBC，需要提供以上信息。

## 入金指南

### 方式一：其他新加坡账户转账

如果你有其他新加坡银行账户（如 iFast），可以直接通过 FAST/PayNow 转账到 OCBC。

- FAST：即时到账，免费
- PayNow：即时到账，免费

### 方式二：国际电汇

1. 从大陆银行购汇（5万美元额度内）
2. 填写电汇单，收款信息：
   - Bank: OCBC Bank
   - SWIFT: OCBCSGSG
   - Account: 你的 OCBC 账号
3. 通常 3-5 个工作日到账
4. 手续费：大陆银行电汇费 + OCBC 入账费

### 方式三：Wise 转账

1. 用 Wise SGD 账户转账到 OCBC
2. 通过 PayNow 或 FAST
3. 通常即时到账，费用低

## OCBC 360 账户与实体卡

### OCBC 360 账户

开户后默认开通的是 **OCBC 360 Account**，这是一个综合储蓄账户：

| 特点 | 说明 |
|------|------|
| 多币种 | 支持 SGD、USD 等 |
| 无最低余额要求 | 普通账户 |
| 利率 | 满足条件可享受 bonus 利息 |

### 申请实体卡

1. 登录 OCBC Digital App
2. 进入 "Cards" 页面
3. 申请 360 Debit Card
4. 选择邮寄地址（可邮寄到国内地址）
5. 等待收卡（通常 1-2 周）

### 实体卡用途

- 全球 Visa/Mastercard 网络消费
- ATM 取现
- 线上支付

## 常见问题

### Q: 开户需要多少钱？
A: **开户免费，无最低存款要求**。但建议入金至少几百新元激活账户。

### Q: 没有新加坡地址能开户吗？
A: **可以**。使用国内地址作为邮寄地址即可。部分用户反馈使用泰国/马来西亚地址也成功开户。

### Q: 护照 NFC 读取失败怎么办？
A: 尝试以下方法：
1. 确保手机 NFC 功能已开启
2. 移除手机壳（部分手机壳阻挡 NFC）
3. 将手机背面中心对准护照封面中心
4. 上下小幅度移动，找到最佳感应点
5. 保持 5-10 秒不动
6. 换一部手机重试

### Q: 审核被拒怎么办？
A: 可能原因：
- 护照信息读取不完整
- 地址信息不一致
- 触发风控
建议重新申请，确保信息一致。

### Q: OCBC 和 Wise 选哪个？
A: | 维度 | OCBC | Wise |
|------|------|------|
| 性质 | 传统银行 | 金融科技公司 |
| 开户难度 | 低（需护照+NFC） | 极低（身份证即可） |
| 实体卡 | ✅ 有 | ✅ 有 |
| 本地账户 | 新加坡为主 | 40+ 国家 |
| 跨境转账费率 | 较高 | 极低 |
| 存款保险 | ✅ 有 | ❌ 无 |

> 💡 **建议两者互补**：OCBC 作为新加坡本地银行账户，Wise 作为跨境转账工具。

### Q: 可以开多个 OCBC 账户吗？
A: 通常每人只能开一个个人账户。如需更多账户，可考虑同时开 Wise、iFast 等作为补充。
