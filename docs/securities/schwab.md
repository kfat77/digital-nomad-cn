## 开户流程详解

**⚠️ 重要提醒：进入页面后，千万不要切换成中文！** 切换中文会自动跳转到嘉信香港，那边要求本人去香港面签，非常麻烦。

**正确标志**：左上角写着 "Charles Schwab International"，且页面是全英文的。

**官方入口**：[international.schwab.com/open-account-intro](https://international.schwab.com/open-account-intro)

### 第一步：进入开户页面

1. 访问 international.schwab.com/open-account-intro
2. 确认页面显示 "Charles Schwab International"（全英文）
3. 往下滑，点击 "Individual Account" 进入开户流程

> ⚠️ 页面跳转可能有延时，耐心等待，不要点回退。

### 第二步：填写个人基本信息

| 字段 | 填写说明 |
|------|---------|
| First Name | 名，拼音大写 |
| Last Name | 姓，拼音大写 |
| Citizenship | China |
| Date of Birth | 出生日期（MM/DD/YYYY） |
| Email | 邮箱 |
| Phone | 大陆手机号，Country code: 86 |
| Passport Number | 护照号 |
| Passport Expiration | 护照过期时间 |
| Passport Country | China |

> 💡 选填项（Optional）都可以跳过不填。

### 第三步：居住地址

**重要**：地址必须如实填写，且与地址证明上的地址保持一致。

1. 使用谷歌翻译将中文地址转为英文
2. 有三行 Street Address：
   - 第一行：从"区"开始的地址（如：Xuhui District, No. 123 XXX Road）
   - 第二行：如有补充地址
   - 第三行：如有补充地址
3. **Country 和 City 不要填在 Street Address 里**
4. City：城市名
5. Country：China

> ⚠️ 这里的地址一定要和地址证明（信用卡账单/水电单）上的地址一致！

### 第四步：设置账户

1. **账户用途**：选 "General purposes"（一般投资）
2. **资金来源**：选 "Salary, wages or savings"
3. **是否被雇用**：如实填写
   - 如果在职：填写公司行业、公司名和地址
   - 如果自由职业：如实说明

### 第五步：背景调查与交易权限

1. **背景调查**：全部选 No
2. **无纸化文件**：全部勾选 Yes
3. **融资账户（Margin）**：建议选 Yes（可先开通，后续按需使用）
4. **期权交易（Options Trading）**：建议选 Yes
5. **Thinkorswim 交易平台**：**一定要选 Yes**

> 💡 Thinkorswim 是嘉信从 TD Ameritrade 收购的顶级交易软件，拥有更强大的图表分析、期权交易工具和实时见解，强烈建议开通。

### 第六步：期权策略级别选择

| 级别 | 权限 | 适合人群 |
|------|------|---------|
| Level 0 | 备兑（Covered） | 新手/稳健型 |
| Level 1 | 做多（Long） | 基础期权投资者 |
| Level 2 | 价差（Spreads） | 进阶型 |
| Level 3 | 裸期权（Short Uncovered） | 高风险专业投资者 |

> 💡 新手建议选择 Level 0 或 Level 1。级别越高，对交易经验和财务状况审核越严格。

### 第七步：收入和投资经验调查

如实填写：
- 年收入范围
- 净资产范围
- 投资经验（年限）
- 投资目标

### 第八步：确认与协议

1. 核对所有信息
2. 勾选同意各种开户协议
3. 电子签名

### 第九步：上传文件

| 文件 | 要求 |
|------|------|
| 护照 | 清晰照片或扫描件 |
| 地址证明 | 近 3 个月账单（信用卡/水电/煤气） |

> ⚠️ 文件名必须是英文。上传可能失败，按提示重试即可。

### 第十步：提交审核

- 提交后进入审核阶段
- 账户可以正常登录，但**不能交易**
- 审核通过后会收到邮件通知

## W-8BEN 表格填写

W-8BEN 是美国税务居民身份声明表，**所有非美国投资者都必须填写**。这是开户的必经之路，也是享受中美税收协定优惠税率的关键。

### 为什么要填 W-8BEN？

| 不填 | 填写 |
|------|------|
| 美国政府按最高税率预扣税款（30%） | 享受中美税收协定优惠税率（股息税 10%） |
| 投资收益被大幅削减 | 合法合规地最大化投资收益 |

> 💡 W-8BEN 是美国政府发给外国投资者的「税务通行证」。填了它，你赚的钱才不会被美国政府多扣税。

### 填写要点

1. **账户持有人信息**：选择 "Individual"（个人）
2. **国籍/税务居民国**：China
3. **美国公民身份调查**：**全部选 No**
4. **税务识别号**：填写中国身份证号
5. **税收协定优惠**：勾选中美税收协定

> ⚠️ 嘉信不走 CRS，但 W-8BEN 是 IRS（美国国税局）要求的税务声明。两者是不同体系——W-8BEN 解决的是美国预扣税问题，CRS 解决的是全球账户信息交换问题。

## 入金指南

### 推荐入金路径

| 路径 | 手续费 | 时间 | 推荐度 |
|------|--------|------|--------|
| **香港银行卡 → 嘉信** | 低 | 1-3 天 | ⭐⭐⭐⭐⭐ |
| **Wise → 嘉信** | 低 | 1-3 天 | ⭐⭐⭐⭐⭐ |
| 电汇（大陆银行） | 高 | 3-5 天 | ⭐⭐⭐ |

### 香港银行卡入金

1. 登录嘉信官网，获取 USD 入金账户信息（ACH 路由号 + 账号）
2. 登录香港网银（如汇丰/中银）
3. 发起国际汇款，填写嘉信的 USD 账户信息
4. 通常 1-3 个工作日到账

### Wise 入金

1. 在 Wise 中持有 USD 余额
2. 通过 Wise 的美国本地账户（ACH）转账到嘉信
3. 通常 1-2 个工作日到账
4. 费用极低

## 常见问题

### Q: 嘉信真的不走 CRS 吗？
A: **是的**。嘉信理财是美国本土券商，目前不参与 CRS（共同申报准则）信息自动交换。这意味着你的账户信息不会被自动交换给中国税务机关。但请注意：
1. 你仍需依法申报境外投资收益
2. 如果主动要求或司法协助，信息仍可能被提供
3. 政策可能变化

### Q: 开户审核需要多久？
A: 通常 1-5 个工作日。审核期间可以登录账户查看状态，但不能交易。

### Q: 没有港卡可以入金吗？
A: **可以**。除了港卡，还可以通过 Wise、大陆银行电汇等方式入金。但港卡和 Wise 是最方便、成本最低的方案。

### Q: 嘉信和盈透（IBKR）选哪个？
A: | 维度 | 嘉信 | 盈透 |
|------|------|------|
| 美股佣金 | 免费 | $0.0035/股 |
| 全球市场 | 美、加、欧、日等 | 150+ 市场 |
| 闲置资金利息 | 一般 | 3%+ |
| 最低资金 | 无 | 无 |
| CRS | 不走 | 走 |
| 交易工具 | thinkorswim 强 | TWS 专业 |
| 适合 | 美股长期投资者 | 全球市场交易者 |

> 💡 如果** primarily 投资美股/ETF，选嘉信**。如果需要交易全球市场、期权、期货，选盈透。两者可以互补。

### Q: 嘉信安全吗？
A: 非常安全：
- 纳斯达克上市公司（$SCHW$），财报全透明
- FINRA 和 SEC 资深成员
- SIPC 保险（最高 $500,000）
- 即便券商倒闭，客户资金也有兜底

### Q: 可以用大陆银行卡直接入金吗？
A: **可以但不推荐**。大陆银行电汇到嘉信手续费高（约 $20-50/笔），且可能被银行询问用途。建议通过港卡或 Wise 中转。
