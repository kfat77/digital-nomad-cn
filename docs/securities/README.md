## ⚠️ 重要提醒

1. **外汇管制**：中国实行外汇管制，个人每年便利化购汇额度为等值 5 万美元。超出部分需合规申报。
2. **税务合规**：境外投资收益需依法申报个人所得税。境外券商可能要求填写 W-8BEN 表格（美国税务居民身份声明）。
3. **投资风险**：海外投资涉及汇率风险、市场风险等，请充分了解后决策。
4. **CRS**：共同申报准则下，境外金融机构可能向中国税务机关交换账户信息。
