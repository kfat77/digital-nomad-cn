## 最新政策变化

2024年11月1日，香港优才计划政策突变：

| 变化 | 旧政策 | 新政策 |
|------|--------|--------|
| 评分方式 | 逐项计分（满分245分，80分及格） | 12项是非题，满足6项即可 |
| 配额 | 2023-2024年无配额限制 | 依然无配额限制 |
| 审批周期 | 9个月左右 | 预计缩短3个月 |
| 语言要求 | 基本资格之一 | 纳入评核标准范畴 |
| 续签 | 3+3+2模式 | 3+3+2，新增"稳定收入"要求 |

## 新版打分标准（12项评核准则）

满足 **12项中至少6项** 即可申请：

1. **年龄**：50岁或以下
2. **学历（一）**：合资格大学的硕士或博士学位
3. **学历（二）**：合资格大学STEM（科学、技术、工程、数学）专业的硕士或博士
4. **语言能力（一）**：具备两种语言能力（如中文+英语，或中文+其他外语）
5. **语言能力（二）**：具备良好的英文书写及口语能力（雅思≥7 / 托福≥94 / 六级≥520）
6. **工作经验（一）**：5年或以上学位程度的工作经验
7. **工作经验（二）**：3年或以上跨国公司/知名企业工作经验
8. **工作经验（三）**：3年或以上创新科技、金融、国际贸易行业经验
9. **国际经验**：2年或以上国际工作经验（原居国家/地区以外）
10. **全年收入**：申请前12个月税前收入达100万港元或以上
11. **业务所有权（一）**：拥有年盈利500万港元以上公司的控制权（≥50%股权）
12. **业务所有权（二）**：拥有一家上市公司10%或以上股权

> 💡 **提示**：新版打分制度更倾向英语能力强的国际型人才、高收入企业家，以及金融/创新科技/国际贸易特定行业人才。

## 申请流程

10步搞定香港身份：

```
01. 评估资格 → 对照12项准则，看是否满足6项
02. 准备资料 → 个人资料、工作证明、学历证明、赴港计划书等（1-2个月）
03. 递交资料 → 网上递交，当天获得档案号
04. 甄选环节 → 入境处审核，约6-9个月
05. 收到获批函 → 邮件通知
06. 办理香港身份证 → 提供赴港同意书 → 办理逗留签 → 激活 → 预约办身份证
07. 维护香港身份 → 保持与香港的联系（生活/工作轨迹）
08. 续签 → 首签3年，续签模式3+3+2或3+5（顶尖人才）
09. 转永居 → 7年后申请香港永久居民
```

---

## 续签要求

- **续签模式**：3+3+2（一般人才）或 3+5（顶尖人才）
- **关键要求**："定居" + "就业/创业" + **新增"稳定收入"**
- **通常居住**：合法、自愿、以定居为目的在港居住（如读书、工作），即使短暂离港仍视为通常居住
- **考量因素**：在港工作、子女在港就读、受雇香港公司、两地频繁往来等

## 是否需要中介

| 情况 | 建议 |
|------|------|
| 时间成本高、无暇研究 | 建议找中介，提升效率与成功率 |
| 材料复杂、履历特殊 | 中介可提供个性化方案 |
| 需续签+永居规划 | 中介提供全流程服务 |
| 条件清晰、有时间 | 可自行申请，官网入口递交 |

## 适合人群

以下三类人群更具优势：

1. **名企背景或海外经验的高管/技术人才** — 年龄、语言、工作经验、名企经历可累计多项得分
2. **名校理工科硕博毕业生** — 学历+STEM专业可拿2分，结合名企/海外经验更易达标
3. **企业家或公司股东** — 新设"业务所有权"项，企业主/上市公司股东（≥10%股份）有优势
