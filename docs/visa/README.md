# 签证模块概述

面向中国护照持有者的全球签证攻略，重点覆盖打工度假签证（WHV）、数字游民签证、长期工作签证等路径。提供完整的申请条件、流程、材料清单及实操经验。

> 本模块内容参考了多位成功申请者的真实经验，以及移民中介、律师的公开信息。所有内容仅供学习参考，不构成移民或法律建议。具体政策以各国移民局官网为准。

## 核心提醒

1. **一生仅一次**：澳洲WHV（462）和新西兰WHV均一生只能申请一次，准备充分再出手
2. **提前3个月准备**：语言成绩、资金证明、公证等材料耗时较长
3. **关注官网**：政策每年微调，以移民局官网为准
4. **DIY可行**：官方流程清晰，无需中介，可省下数万元费用
