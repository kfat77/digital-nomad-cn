# 终极目标：Chinese Global Mobility Open Infrastructure

## 定义

```
Chinese Global Mobility Open Infrastructure（中国人的全球流动开放基础设施）

是一个由全球华人社区共建、开源、数据驱动的全球流动信息基础设施，
为每一个中国人（以及未来扩展到全球所有人）提供：

1. 全球国家/城市/签证的完整数据
2. AI 驱动的智能决策工具
3. 开放 API 和 MCP 接口
4. 实时更新的全球流动数据网络
5. 社区自治的可持续治理模式

使命：让全球流动信息像水电一样普及
愿景：成为全球流动的默认基础设施
价值观：开放、准确、实用、社区驱动
```
