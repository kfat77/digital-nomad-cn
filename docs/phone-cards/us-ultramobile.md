# 🇺🇸 美国 Ultra Mobile 保号方案

> Ultra Mobile 是美国 T-Mobile 旗下的 MVNO 运营商，以其极低的保号成本（$3/月）成为许多需要美国手机号用户的首选。

## 基本信息

| 项目 | 内容 |
|------|------|
| 月费 | $3/月（PayGo 套餐）|
| 包含 | 100分钟通话 + 100条短信 |
| 流量 | 无（需额外购买）|
| 国内收短信 | ✅ 支持（需开启 WiFi Calling）|

## 获取方式

### 方式一：官网购买
- 访问 ultramobile.com
- 选择 PayGo 套餐
- 可邮寄到转运地址

### 方式二：淘宝/闲鱼
- 搜索「Ultra Mobile 激活卡」
- 价格约 50-100 元
- 已预装 PayGo 套餐

## WiFi Calling 设置（国内使用）

开启 WiFi Calling 后，在中国境内可以通过 WiFi 网络接打电话和收发短信，相当于使用美国本地网络。

### iPhone 设置
1. 设置 → 电话 → WiFi 通话 → 开启
2. 设置地址（美国任意地址即可）
3. 连接 WiFi 后即可使用

### Android 设置
- 设置 → 网络 → WiFi 通话 → 开启
