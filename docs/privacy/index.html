隐私政策

最后更新：2026年7月8日

1. 我们不收集什么

数字游民指南是一个开源静态网站。默认情况下，我们不使用 Cookie、不收集个人身份信息、不存储 IP 地址、不跟踪你的跨站行为。

2. 我们收集什么

如果你选择启用可选的分析功能，我们会收集以下匿名化数据：

  • 访问的页面路径（不含个人身份信息）
  • 页面标题和语言设置
  • 屏幕分辨率（用于优化体验）
  • 时区（用于按地区分析流量）
  • 匿名会话 ID（24小时内失效）

3. 本地数据

我们使用浏览器的 localStorage 仅在你的设备上存储以下数据：

  • 主题偏好（深色/浅色模式）
  • 你浏览过的页面统计（用于"你的足迹"功能，可随时清除）

这些数据永远不会离开你的浏览器，除非你在隐私设置中明确配置第三方分析端点。

4. 第三方分析（可选）

我们提供与第三方隐私优先分析工具（如 GoatCounter、Plausible）的集成能力。这些工具：

  • 不使用 Cookie
  • 不收集个人身份信息
  • 不追踪跨站行为
  • 不将数据出售给第三方

第三方分析默认完全关闭，直到你在配置中手动启用。

5. 你的权利

  • 随时清除本地数据：
  • 启用浏览器的"不跟踪"（Do Not Track）功能，自动禁用所有分析
  • 查看我们持有的关于你的数据：查看你的足迹

6. 开源与透明

本项目的分析代码完全开源。你可以查看 analytics.js 了解具体实现。
