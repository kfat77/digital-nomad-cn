数据端点

所有数据端点均为静态 JSON 文件，由 GitHub Actions 在每次提交时自动生成。可直接通过 HTTP GET 获取。

📊 国家数据

GET https://kfat77.github.io/digital-nomad-cn/api/countries.json

获取 60 个国家的完整结构化数据，包括生活成本、安全指数、网速、签证政策、评分等 20+ 字段。

关键字段：id, name, name_en, region, digital_nomad_score, cost_of_living_index, safety_index, internet_speed_avg_mbps, rent_index, hdi, gdp_per_capita_usd, china_passport_visa_status, has_digital_nomad_visa, coordinates, currency, languages, timezone, population, area_km2, tags

🏙️ 城市数据

GET https://kfat77.github.io/digital-nomad-cn/api/cities.json

获取 122 个数字游民城市的完整数据，包括联合办公、气候、英语普及率、社区规模等。

关键字段：id, name, name_en, country_id, country_name, digital_nomad_score, cost_of_living_index, rent_index, safety_index, internet_speed_avg_mbps, coworking_spaces, coworking_score, climate, avg_temp_c, rainy_season, english_level, chinese_community_size, neighborhoods, food_scene, nightlife, transportation, walkability, bike_friendly, description

🛂 签证数据

GET https://kfat77.github.io/digital-nomad-cn/api/visas.json

获取 60 个国家的签证类型详情，包括数字游民签证、旅游签证、工作签证等。

关键字段：id, country_id, country_name, visa_types (array), updated_at, source

📈 统计摘要

GET https://kfat77.github.io/digital-nomad-cn/api/stats.json

快速获取数据集统计摘要，包括平均值、按地区分布、按签证状态分布等。

AI Agent 使用示例

以下是一个 AI Agent 调用本数据 API 的示例。不需要 API Key，不需要认证，直接 GET 即可。

# 示例：获取所有国家数据并筛选高评分目的地 fetch('https://kfat77.github.io/digital-nomad-cn/api/countries.json') .then(r => r.json()) .then(data => { // 筛选数字游民评分 >= 80 的国家 const topCountries = data.data .filter(c => c.digital_nomad_score >= 80) .sort((a, b) => b.digital_nomad_score - a.digital_nomad_score); console.log(topCountries.map(c => `${c.name}: ${c.digital_nomad_score}分`)); });
# 示例：获取统计摘要 fetch('https://kfat77.github.io/digital-nomad-cn/api/stats.json') .then(r => r.json()) .then(stats => { console.log(`平均数字游民评分: ${stats.countries.avg_digital_nomad_score}`); console.log(`免签国家数: ${stats.countries.by_visa_status.visa_free || 0}`); });
