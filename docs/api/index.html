数据端点

所有数据端点均为静态 JSON 文件，由 GitHub Actions 在每次提交时自动生成。可直接通过 HTTP GET 获取。

📊 国家数据 v1.2

GET https://kfat77.github.io/digital-nomad-cn/api/countries.json

获取 60 个国家的完整结构化数据，包括生活成本、安全指数、网速、签证政策、深度指南字段等 25+ 字段。

核心字段：id, name, name_en, region, digital_nomad_score, cost_of_living_index, safety_index, internet_speed_avg_mbps, china_passport_visa_status, has_digital_nomad_visa, climate_zone, timezone, dn_community_size, median_stay_months

深度字段 (v1.2)：healthcare_quality, english_proficiency, monthly_rent_usd_low/high, food_cost_usd_daily, transport_cost_usd_monthly, coworking_spaces_count, local_tax_rate, pros, cons, ideal_for, best_neighborhoods

🏙️ 城市数据

GET https://kfat77.github.io/digital-nomad-cn/api/cities.json

获取 122 个数字游民城市的完整数据，包括联合办公、气候、英语普及率、社区规模、远程工作友好度等。

关键字段：id, name, name_en, country_id, country_name, digital_nomad_score, cost_of_living_index, rent_index, safety_index, internet_speed_avg_mbps, coworking_spaces, coworking_score, climate, avg_temp_c, rainy_season, english_level, chinese_community_size, neighborhoods, food_scene, nightlife, transportation, walkability, bike_friendly, description, remote_work_friendly_score, monthly_cost_usd_low, monthly_cost_usd_high

🛂 签证数据

GET https://kfat77.github.io/digital-nomad-cn/api/visas.json

获取 60 个国家的签证类型详情，包括数字游民签证、旅游签证、工作签证等。

关键字段：id, country_id, country_name, visa_types (array), updated_at, source

代码示例

无需 API Key，直接 GET 即可。以下示例展示如何调用数据 API。

// 获取所有国家数据并筛选高评分目的地 fetch('https://kfat77.github.io/digital-nomad-cn/api/countries.json') .then(r => r.json()) .then(data => { // 筛选数字游民评分 >= 80 且生活成本 <= 60 的国家 const topCountries = data.data .filter(c => c.digital_nomad_score >= 80 && c.cost_of_living_index <= 60) .sort((a, b) => b.digital_nomad_score - a.digital_nomad_score); console.log(topCountries.map(c => `${c.name}: ${c.digital_nomad_score}分`)); });
// 获取统计摘要 fetch('https://kfat77.github.io/digital-nomad-cn/api/stats.json') .then(r => r.json()) .then(stats => { console.log(`平均数字游民评分: ${stats.countries.avg_digital_nomad_score}`); console.log(`免签国家数: ${stats.countries.by_visa_status.visa_free || 0}`); });
